## 【业务拆解】

### 一、项目背景与核心认知

- 名义上是"抖音自然流CPS"项目，实际上390万GMV基本都在B站跑出
- 核心认知：CPS才是本质，抖音只是平台，自然流只是手段
- 双11同类玩法曾创造23天GMV破千万、利润7万

### 二、核心玩法：红包+投流

1. **红包作为钩子**：通过红包优惠锁定用户下单行为，降低选品差异化对转化的影响
2. **投流作为放大器**：突破自然流流量天花板，精准触达潜在消费人群

### 三、投流策略迭代（关键决策）

| 策略 | 做法 | 结果 |
|------|------|------|
| 初始策略 | 投蓝链点击（预期精准流量高转化） | 蓝链成本2元+/点击，盈利临界点1元以下，2天后放弃 |
| 优化策略 | 改投播放量（成本低） | 成本可控，虽精准度低但不亏损 |
| 密令成交 | 引导用户走密令链路 | 占比2/3，可获平台激励 |

### 四、成交链路博弈

- 密令成交占2/3，蓝链成交占1/3
- 核心原因：通过视频内容、评论区话术、置顶留言多维度引导
- 关键洞察：用户行为是可以被引导的

### 五、投流试错教训

- 扫地机器人视频投流蓝链：300元/蓝链点击，600元买2个点击
- 启示：不同品类投流逻辑不同，不可直接套用经验
- 需要：小预算、多维度测试

### 六、自然流方法论

1. 内容结构偏广告结构而非Vlog结构
2. 脚本骨架：痛点开场→给结果→证据→购买理由→强引导话术
3. 爆点三要素：人群明确、痛点尖锐、承诺具体
4. 核心是批量测试（每品10-30条方向测试），不是押宝单条
5. 自然流跑不通的内容不要投流放大

### 七、付费投流四阶段

1. **冷启动**：小预算多素材多定向，筛素材
2. **跑通**：找到成本稳定的素材作为主力
3. **放量**：预算逐步加，放量的是素材模型+人群模型
4. **稳定**：维持素材上新节奏，用数据看疲劳

### 八、流量闭环

自然流测试爆点 → 付费流放大 → 数据反哺内容迭代 → 找到好模板拍100遍

### 九、B站投流确定性规律

1. 卖货优先投蓝链点击（算法匹配内容+商品+点击意愿三类用户）
2. 普通蓝链测试成本 → 放心投设置稍低价格跑量
3. 短时大量流量：提前2.5小时投6小时档位
4. 人群定向：排除18岁以下，聚焦发达区域+男性，用自定义人群包

### 十、品类选择与长期发展

- 数码：热度高但佣金率低（京东0.01事件后更惨淡）
- 家电：佣金率远高于数码，长期更优
- 潜力：美妆（高佣金，竞争少，待探索）
- 放大路径：投流放量→矩阵化→团队化→AI提效

## 【可迁移做法】

1. CPS核心是按成交付费，平台和流量形式只是手段
2. 投流策略必须快速试错、果断调整（蓝链2天不达标即切换）
3. 密令成交可获平台激励，通过引导提高占比
4. 自然流→付费流闭环：先验证后放大
5. 素材工程化生产，不依赖灵感
6. 投流3天滞后性，及时止损
7. 找到好模板拍100遍，不费力拍好100条
