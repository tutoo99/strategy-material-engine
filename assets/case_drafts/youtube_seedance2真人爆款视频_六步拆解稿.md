## 第四步：第三遍细读

这一轮专抓“十字路口”和“动作”。

### 1. 决策点：入局 YouTube 时，依赖现成 SOP 还是先练爆款复刻基本功？
- **选择：** 不等 SOP，先把真人 AI 爆款复刻当成基本功
- **依据：** 作者认为 YouTube 和 AI 变化太快，平台不会长期允许“有明确 SOP 后很多人一起吃红利”
- **动作：**
  1. 先接受“没有标准答案”的前提
  2. 把自己的目标拆成 IP 判断、脚本理解、分镜拆解、生成、剪辑这些能力
  3. 从真人 AI 爆款复刻切入训练

### 2. 决策点：爆款 IP 从哪里找？
- **选择：** 梗百科账号 + YouTube 无痕首页 + 全球电影/游戏资讯三路并行
- **依据：** 国内火的不一定国外火，以前火的不一定现在还火，长视频有量的不一定 Shorts 也有量
- **动作：**
  1. 去 B 站/抖音关注梗百科账号补热梗
  2. 打开无痕模式进入 YouTube 首页
  3. 先点 `Shorts`
  4. 等右侧短视频完全加载
  5. 再点回首页持续刷新
  6. 看到陌生形象就截图，用 Google 以图搜图或问 AI 确认 IP 名称

### 3. 决策点：发现热门 IP 后，是直接做还是先验证？
- **选择：** 先验证再做
- **依据：** 作者举例说明，英雄联盟在印象里流量很大，但当月 Shorts 并没有特别高的播放
- **动作：**
  1. 打开 YouTube 搜 IP 关键词
  2. 点击 `过滤`
  3. 依次选择 `热门程度` → `本月` → `短视频`
  4. 看近几天是否还有千万或大几百万播放
  5. 点进相关视频，右上角三个点 → `说明`
  6. 点击蓝色标签，看同标签视频池

### 4. 决策点：围绕 IP 做内容时，新人先做哪一层？
- **选择：** 先做第一层外壳套用和第二层轻度二创
- **依据：** 作者明确说大部分 AI 创作者做到前两层就够了，第三层门槛太高
- **动作：**
  1. 【推测】打开同一 IP 的多条爆款视频做对比
  2. 【推测】把样本分成“外壳套用 / 轻度二创 / 深度演绎”三层
  3. 【推测】先选择前两层样本开做，再视数据升级到第三层

### 5. 决策点：爆款脚本从哪里找？
- **选择：** YouTube 自动推荐为主，国内平台反向补充
- **依据：** YouTube 真人账号容易被抄烂，只看油管容易同质化
- **动作：**
  1. 准备一个没有关注过其他赛道的小号
  2. 关注作者列出的真人对标账号
  3. 连续刷这些账号的视频
  4. 回到首页不断刷新，喂出相似推荐
  5. 在抖音/快手/小红书/B站搜 `搞笑`、`反转`、`欧亨利式结尾`
  6. 点击视频标签顺藤摸瓜继续找脚本源

### 6. 决策点：什么脚本值得拿来做 AI 复刻？
- **选择：** 强画面、弱文案、跨文化可理解，并经过 4 条红线筛查
- **依据：** 靠语言包袱的脚本跨语言转换会失真；强文化背景的梗老外不一定懂；违规内容还会影响 YPP
- **动作：**
  1. 拿到脚本后先判断是不是靠画面和情绪推进
  2. 检查四条红线：未成年人危害、恶心擦边、文化门槛过高、画面不强
  3. 不合格脚本直接淘汰或重写

### 7. 决策点：分镜脚本用什么工具拆？
- **选择：** 用 Gemini AI Studio 拆，但只把它当草稿器
- **依据：** Gemini 免费且强，但会乱加镜头、搞错角色关系和运镜
- **动作：**
  1. 打开 `https://aistudio.google.com/welcome`
  2. 选择 `Gemini 3.1 Pro Preview`
  3. 粘贴对标 YouTube 视频链接和三幕式提示词模板
  4. 先让它输出“故事内核架构”
  5. 确认没跑偏后回复“继续”
  6. 生成完整分镜表后逐镜对标原视频修正

### 8. 决策点：角色参考图怎么生成，遇到版权/人脸问题怎么办？
- **选择：** 先用 Gemini 生成三视图；版权问题换服装；人脸问题转彩色素描
- **依据：** 高辨识度动漫服装和真人面孔都可能卡生成或限制
- **动作：**
  1. 找角色全身照
  2. 打开 `https://gemini.google.com`
  3. 选择 `Create image`
  4. 上传参考图
  5. 输入三视图提示词，要求全身、纯白背景、真人实拍风格
  6. 如遇版权，转去豆包改服装
  7. 如遇人脸问题，转彩色素描图再生成

### 9. 决策点：Seedance 2.0 和 1.0 怎么分工？
- **选择：** Seedance 2.0 负责主流程，Seedance 1.0 负责缺镜或无台词补镜
- **依据：** Seedance 2.0 支持口型同步但有 `10 镜 / 15 秒` 限制，1.0 更适合补镜
- **动作：**
  1. 按每 10 个分镜截一张图
  2. 打开 Seedance 2.0 上传角色图和分镜图
  3. 输入要求“写实镜头、有音效、不要 BGM、不要字幕”
  4. 发现缺镜后回到 Gemini 生成缺失关键帧
  5. 用 Seedance 1.0 单独补无台词镜头

### 10. 决策点：剪辑节奏和 BGM 怎么处理？
- **选择：** 优先复用原 BGM，再按自己镜头节奏重剪
- **依据：** 作者强调 BGM 是情绪驱动核心；AI 复刻不能完全秒级同步原片
- **动作：**
  1. 在 YouTube 原视频下方尝试识别 BGM
  2. 识别不到就用 QQ 音乐听歌识曲或问 Gemini
  3. 在剪映先铺 BGM
  4. 信息镜头剪到约 0.5 秒
  5. 情绪镜头保留 1-1.5 秒
  6. 去剪映音效库搜场景词，比如 `车撞`

## 第五步：跳出细节，提炼心法

这一轮不回原文，只看前面整理出的身份证和决策地图。

### 作战三原则
1. 做任何 IP 之前，必须走完 `搜索 → 过滤 → 热门程度 → 本月 → 短视频` 的验证链路，只做当前还有量的 IP。
2. AI 生成的分镜永远只算草稿，必须逐镜对标原视频，人工修正人物关系、镜头方向、情绪过渡和缺失镜头。
3. 先做出第一条成片，再围绕数据和平台反馈迭代，不等“全懂了”再开始。

### 最大一个坑
- **坑点：** 看到一个热门形象或故事就直接开做，没有先验证流量，也没有先筛脚本，容易零播放并浪费生成成本。
- **解决方案：** 把“IP 当月流量验证 + 脚本四条红线筛查”做成前置闸门，没过闸不进入后面的分镜、生成和剪辑。

### 最值钱一句忠告
- 不要猜什么火，让平台告诉你什么火。先花 2 小时像普通用户一样刷 YouTube 无痕首页，把平台主动推给你的热门 IP 和爆款脚本截图存档，这就是免费的选题池。
