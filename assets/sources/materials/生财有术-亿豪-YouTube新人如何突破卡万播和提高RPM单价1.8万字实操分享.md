# 二、 新人如何突破卡万播



下面就进入今天的第一个主题：<span style="color: rgb(143,149,158); background-color: rgb(187,191,196)"> </span>新人如何突破卡万播



首先在的我认知中，所谓的「新人」定义为，做Youtube视频，1-5个月，就是你从来没做过视频，到开始做，然后做5个月！当然对于生财优秀的圈友们，肯定有人中龙凤！



回到我们的主要问题「新人卡万播」我感觉最主要的有几个点：

1. 抛开脚本不谈，你做的视频质量很差，整个视频多个镜头都是崩坏的&#x20;

2. 好的脚本，制作是平均水平，但是开头没做好&#x20;

3. 好的脚本，开头做好了，视频也制作好了，但是这个脚本做的人太多了&#x20;

4. 脚本不好，制作的很好，也卡万播

5. 限流的可能



当然还有其他原因，我就不一一列举了！
