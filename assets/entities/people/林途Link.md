# 林途Link

## 简介

- 公众号运营的底层逻辑，将IP、流量、变现串成一条从注意力到信任再到交易的线性链路，包含200+公众号矩阵的赛道选择、选题系统、同题异构打法，以及微信生态下一个红利窗口的判断。
- 今天不讲一篇爆文10W+能赚多少钱，不给大家制造焦虑，也不讲普通人如何变现100W。
- 在传统社会一个人的影响力主要来自三件事，身份、资源、人脉，这被称为第一影响力，有明显局限，受家庭背景、职位、地域行业等多重因素限制，多数人的第一影响力并不强，也并不稳定，而IP它让普通人有机会通过内容、知识与价值输出，构建一套完全由自己主导、可积累、可扩大的影响力体系。

## 代表经历

- 今天不讲一篇爆文10W+能赚多少钱，不给大家制造焦虑，也不讲普通人如何变现100W。
- 「公众号3 年、200+ 公众号、几十亿流量，专门帮做公众号流量压着、变现混乱中小的创作者，把账号做成能赚钱的号」
- 主要还是紧扣夜话主题，1主线「公众号」3个点「IP」「流量」「变现」
- 只聊个人站在公众号，内容，以及公众号商业变现和最近微信生态的变化角度发表自己的个人看法，我希望的是这篇文章能够从内容创作以及流量底层，变现逻辑上有一些帮助，但不一定在现在。

## 方法论

- ③ 生态型 IP： 不止是一个人而是外加我自己的资源，链接到我能够链接到我背后的资源， 内容 + 社群 + 方法 + 案例 + 你链接的资源等等。 这就是为什么有的人，明明文笔一般，长期下来变现却最强，这种不可替代性太强。
- 今天不讲一篇爆文10W+能赚多少钱，不给大家制造焦虑，也不讲普通人如何变现100W。
- ① 功能型 IP 这个号教写作的，这个号写搞钱的，比如我的林途Link就是写如何做公众号。 用户记住的是功能，不是人，好处是好起步， 坏处是可替代，市面上那么多教写公众号的为什么要关注我。
- ② 人格型 IP 这个人说话很实在，这个人是一路熬出来的，这个人有一套明确的方法论， 比如我我漏过脸，我俩脾气对，眼缘好等等， 用户记住的是我这个人，信任度就相对很高。
