# 许老师

## 简介

- 哈喽大家好，我是许老师，生财联合办公空间业务负责人，我每天的工作，就是跟联合办公里的一群创业者朝夕相处，聊项目、聊卡点、帮他们把下一步想清楚。和这群圈友日常大量沟通下来，我发现一个非常普遍的现象——大家在做决策的时候太慢了，甚至不敢做决策。
- 如果你还沿用打工时的思维——"这件事有80%的把握我才做"，那你很多决策就会做得非常慢，甚至不敢决策。小排老师在自己的创业年度复盘文里说了一句话，我觉得特别精准："我打工太久了，对不确定性已经完全没有承受能力。现在我是创业者、是公司老板，对不确定性的承受能力，是我的必修课。"我相信这句话会戳中很多从职场出来创业的圈友。
- 当你还在打工时，你不需要做决策，公司给你派任务，你只需要执行，做完了拿绩效，不用承担风险。但当你成为老板，你每天睁开眼就要面临无数个选择：要不要招这个人？要不要投这个广告？要不要去参加这个线下大会？要不要跟这个人谈合作？

## 代表经历

- 但你算一下：玩10次，赢1次赚100，输9次每次亏10块共亏90。10次下来，100减90，还赚10块。每玩10次就稳赚10块钱，那你当然应该一直玩这个游戏。
- 我们大致帮他算一下这笔账。最坏的情况，投了1000块一个水花都没有，损失1000块——这就是亏损上限，非常确定。而一旦投通了呢？他本身自然流阶段一个商单报价就好几千，如果投流让他粉丝翻倍、商单价格再涨一个档，一个月多接一单，一年就是小十万的增量。哪怕投通的概率只有30%，期望值 = 30% × 100000 – 70% × 1000 = 30000 – 700 = 约29300。一笔1000块的支出，期望值接近3万，怎么算都划算。
- 哪怕成功概率只有5%，我们代进公式算一下：赢的部分 = 5% × 10000 = 500，输的部分 = 95% × 0 = 0，期望值 = 500 – 0 = 500。虽然概率只有5%，但每发一条消息的期望值都是正500块——为什么不发呢？
- 只要算出来是正数，哪怕成功率只有10%，这件事也值得做。反过来，如果期望值是负数，哪怕成功率有90%，也不值得做——比如为了赚1万块去冒赔掉100万的风险。

## 方法论

- 先说投流。很多人项目跑通了0到1，靠自然流量每个月赚几万块，然后就一直靠自然流，不敢碰付费流量。一想到要花钱买流量，万一没效果打水漂了怎么办？
- 我自己以前也是一个非常脸皮薄的人，出去参加大会，别人都在那社交，我就呆呆地站在那里半天，不知道干什么。后来我发现如果是这个状态，我的创业项目不会有任何推进——我怎么争取外界的资源帮助我？现在我变了。每次大会我都会主动说："我们下一步需要xxx 的帮助，有没有谁有资源？有的话欢迎介绍给我。"被拒绝是常态，通常十次九次被拒绝，无所谓。
- 要不要花2000块试试投流？纠结了一个月，没投。想找一个行业大佬聊聊，犹豫了一周，什么也没做。项目跑通了想招个助理，又怕万一不好用，5000块工资打水漂。这种状态，说实话我太熟悉了，因为我自己以前也是这样。
- 只要算出来是正数，哪怕成功率只有10%，这件事也值得做。反过来，如果期望值是负数，哪怕成功率有90%，也不值得做——比如为了赚1万块去冒赔掉100万的风险。
