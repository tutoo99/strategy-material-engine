## 三个陷阱

```
陷阱一：创始人永远困在前端
  前三年正常 → 一辈子？→ 没有别的事了
  用户永远从零到一测试 → 你能绑定多久？

陷阱二：来什么流量做什么 = 被客户定义
  无法定义你的客户 → 只能被客户定义
  → 商业认知和系统思考能力不足
  → 第二次第三次出手不从底层开始

陷阱三：不可规模化 = 无法退出
  桃李满天下但永远困在那里
  收入靠不停曝光维持
  → 做商业的目的不是持续在线而是退出未来
```

## 段永平 vs 王石

```
段永平：知道有人做得更好 → 退出做投资 → 财富指针
王石：全世界需要万科 → 鞠躬尽瘁 → 债务缠身
→ 觉得王石爷们但财富指针瞄准段永平
```
