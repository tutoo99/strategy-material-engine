## 知识付费真相

```
你以为的本质          真实的本质
卖知识/卖课    →     投资/筛选/入股
赚客单价       →     持有股份被动收入

罗振宇：得到商学院=筛选好项目→入股
混沌：  商学院=赛马机制→好项目匹配好资金
樊登：  裂变收割现金→投资
生财：  找好项目找IP

共同路径：前端卖课筛人 → 后端投资入股
```
