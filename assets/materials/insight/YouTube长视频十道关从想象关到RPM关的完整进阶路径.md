## 十道关速查

| 关卡 | 核心卡点 | 破解方向 |
|------|----------|----------|
| 1. 想象关 | 不敢尝试 | 找弱画面低成本赛道 |
| 2. 赛道关 | 一次选对执念 | 快速试错多号测 |
| 3. 制作关 | 仿不到位 | 用数据反馈迭代 |
| 4. 挂0关 | 信息差 | 调整策略第二天见量 |
| 5. 数据关 | CTR和留存低 | 优化封面和内容结构 |
| 6. 没量关 | 6种没量模式 | 研究运营策略 |
| 7. 时间关 | 自我怀疑 | 宏大叙事给自己打气也可以 |
| 8. 矩阵关 | 自动化降质量 | 先手搓稳定再提效 |
| 9. 坚持关 | 不知道该不该放弃 | 用数据判断不是凭感觉 |
| 10. RPM关 | 高播放低收益 | 赛道选择时就看RPM |

## 核心认知

- 能意识到问题和意识不到问题是有解和无解的区别
- 挂0关的信息差是分水岭，懂的人第二天就起量不懂的人几个月放弃
- 开通YPP不是终点而是赚美刀入场券，真正长期问题是如何应对没量
- 对视频制作数量要祛魅，一天做100条0播放的内容毫无意义
