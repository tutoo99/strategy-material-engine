# 形式爆款vs脚本爆款的本质区别与选择逻辑

## 对比

| 维度 | 形式爆款 | 脚本爆款 |
|------|----------|----------|
| 镜头数 | 少(1-5个) | 多(二三十个) |
| 时长 | 3-10秒 | 30秒到1分钟+ |
| 脚本 | 无实质故事剧情 | 有故事或剧情 |
| 核心驱动 | 感官冲击+套路 | 故事讲述+信息传递 |
| 门槛 | 低 | 更高 |
| 窗口期 | 短 | 更长 |
| 反馈速度 | 快 | 更慢 |
| 制作成本 | 低 | 更高 |

## 关键洞察

- 门槛越低意味着窗口期越短。Sora2让制作极简化(6分钱/条)，但爆款周期可能只有几天
- AI类形式爆款生命周期约1-2周，但可以通过变种延长
- 选择标准：喜欢创意搞抽象选形式爆款，喜欢写故事设计脚本选脚本爆款
- 每个人的路不一样，没有标准答案
