# 把真实状态变成内容的一部分

## 核心观点
不需要完美人设，真实=信任。

## 四种真实状态的处理
| 状态 | 做法 |
|------|------|
| 身体不舒服 | 写不舒服+承诺尽量写好 |
| 临时没时间 | 写时间少+保证质量 |
| 心力不足 | 写断更原因+分享补充心得力的方法 |
| 压力太大 | 调整节奏+每周一天只发精品 |

## 效果
读者没有抵触，继续看、点赞、评论。真诚比技巧重要。
