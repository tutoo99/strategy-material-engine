# 垂直小号收益核心是信任

## 核心观点
专业→信任→变现，这条链路不需要刻意设计。

## 轻舟40天变现数据
- 生财体验卡：邀请35人，领卡29人，入营15人
- 付费咨询：若干
- 工具推广（帮圈友大臣推批量文生图工具）：4位数收入
- 精准粉丝：2100+

## 收益分层设计
| 人群 | 策略 |
|------|------|
| YouTube新手 | 安利生财，发放体验卡 |
| 有经验YouTuber | 简单问题直接答，复杂问题引导付费咨询 |
| 没接触过YouTube | 有执行力发体验卡，没有的直接拒掉 |

## 底线原则
不相关的广告全部拒绝，不做违背赋能主线原则的事。
