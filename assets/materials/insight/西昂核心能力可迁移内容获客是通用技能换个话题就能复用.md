## 能力迁移模型

```
写笔记的能力 ← 通用 → 选关键词的能力
引导用户的能力 ← 通用 → 理解用户需求的能力
平台规则的理解 ← 通用 → 限流规避的能力

小程序推广 ——→ 移民获客 ——→ 任何高客单价获客
     换个话题就行
```

## 关键对比

| 维度 | 小程序推广 | 移民获客 |
|------|-----------|---------|
| 客单价 | 低 | 高（1万+佣金） |
| 首单时间 | 3个月 | 1个多月 |
| 投入时间 | 更多 | 每天10-20分钟 |
| 能力需求 | 完全相同 | 完全相同 |

## 核心洞察

"有用的东西放出去，该来的人自己就来了。"
"换个方向不用重新来，就是换个话题而已。"
