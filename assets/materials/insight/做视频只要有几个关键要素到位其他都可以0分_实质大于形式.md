# 关键要素到位，其他可以是0分

视频制作拆解：选题——文字脚本——露脸拍摄——拍物品/找素材——整合剪辑/加声音特效——动画建模——发布运营

看上去无从下手。但垃圾堆风格视频：没露脸、几乎没拍摄、没有剪辑、没有声音特效、没有动画建模，就是一个人不停地讲。但就是奏效。

**因为他确实解决了用户的问题。**

做视频不要给自己太大压力，管他什么三秒钟切换镜头（央视标准），普通人30秒切换一次也行，从头到尾不切换也行。
