## 核心公式

```
有核心技术 → 产品不可替代 → 高毛利 → 赚资产
没核心技术 → 产品可替代 → 价格战 → 赚辛苦费

辛苦费 ≠ 资产
辛苦费：今天干明天才有钱
资产：今天建明天还在赚钱
```
