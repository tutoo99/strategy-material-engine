## 求上得中

```
权力系统（管理层）  →  求上：定义核心决策
  ↓
商业系统（商业层）  →  求中：模式+产品+增长
  ↓
执行系统（执行层）  →  求下：纠结宱价定多少

大部分人锁在执行层 → 求下而不得
定一万卖100单 = 定99卖一万单 = 都是100万
后续没设计好 = 前端廉价劳动力
```
