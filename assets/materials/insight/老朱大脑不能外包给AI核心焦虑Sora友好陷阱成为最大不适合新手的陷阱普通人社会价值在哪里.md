## 核心公式

```
AI能力(80分) + 个人基础(60分) = 输出(90分)
AI能力(80分) + 个人基础(10分) = 拖后腿
```

## 陷阱

Sora越友好 → 越省心 → 越不思考 → 越丧失判断力 → 被淘汰
