## 两种社群模式

```
生存模式（老鼠赛跑）           租金模式（价值赛道）
教短周期赚钱项目               一次录制卖无数次
写好路径复制给更多人            经济成本为零
所有人拼死竞争                  底层逻辑/商学院/创新院
过段时间都不赚钱                持续滚动后端超高宱价
路径依赖不突破天花板            好项目好老板好背书

留下原地焦虑的人                留下真正成长的人
本质=项目复制                  本质=知识资产

以平台为核心（AI风口/某平台）   以赛道为核心（垂直深耕）
私域=平台不是赛道              私域=平台不是赛道
```

## 两种出路

```
1. 垂直且深：某个赛道长期围绕某个主题深度建设
2. 大而宽：专家背书+零边际成本租金模式

共同点：最大化降低创始人时间成本 → 做充分有意义的思考
```
