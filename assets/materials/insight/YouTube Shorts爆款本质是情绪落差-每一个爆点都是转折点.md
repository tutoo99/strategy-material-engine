# YouTube Shorts爆款本质是情绪落差

## 核心观点
每个爆点=转折点，反转→情绪落差→故事成功。

## 1.5亿播放视频爆点时间线
| 时间 | 画面 | 情绪 |
|------|------|------|
| 2秒 | 男孩拆爸爸机械腿 | 好奇（为什么） |
| 4秒 | 爸爸发现腿没了 | 好笑+好奇（怎么做） |
| 6秒 | 男孩把腿放进火里 | 继续好奇 |
| 13秒 | 男孩砸金店玻璃 | 震惊 |
| 14秒 | 保安追打男孩 | 紧张 |
| 34秒 | 爸爸找到男孩骂 | 愤怒 |
| 36秒 | 男孩摆出金色腿 | 反转→感动 |

## 规律总结
- 黄金三秒制造悬念
- 每隔几秒一个情绪转折
- 结尾反转回收情绪
- 绝望→希望=最强反转模式
