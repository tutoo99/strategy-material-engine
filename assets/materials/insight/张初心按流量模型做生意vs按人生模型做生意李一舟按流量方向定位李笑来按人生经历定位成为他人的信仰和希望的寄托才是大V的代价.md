## 核心判断

```
流量模型：按流量方向定位 → 紧扣时代 → 但没有信仰 → 永远在追
人生模型：按人生经历定位 → 活成信仰 → 用户自发共创 → 躺赚

大V的代价 = 成为信仰的寄托
很多IP的拧巴 = 想要大流量但承接不住期待

增长 = 12维核心决策系统（角色/位置/资源/节奏...）
不是 = 裂变+做流量+切用户+放大规模
```
