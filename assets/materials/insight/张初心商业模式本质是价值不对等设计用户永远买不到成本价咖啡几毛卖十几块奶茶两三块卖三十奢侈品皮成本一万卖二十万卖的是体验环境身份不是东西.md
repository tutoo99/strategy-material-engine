## 价值不对等设计

```
成本            售价            卖的是什么
咖啡几毛    →    十几块    →    体验/环境/感觉
奶茶两三块  →    三十块    →    感受
皮一万      →    二十万    →    身份
盲盒几十块  →    几千几万  →    设计/稀缺/情绪/认同

公式：成本逻辑（你在卖什么） ≠ 意义逻辑（用户买什么）
```
