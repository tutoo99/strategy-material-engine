## 团队化正循环

```
高客单+高利润+高复购
→ 有空间分钱
→ 导师收入稳定（与结果挂钩）
→ 导师稳定 → 交付质量高
→ 客户结果好 → 续费率高
→ 现金流稳定 → 导师粘性强
→ 正循环 ↑
```

## 业务分工

| 角色 | 职责 |
|------|------|
| 本人 | 流量 + 信任 + 销售 |
| 系统/团队 | 交付 |
| 内容/分发 | 获客 |

**核心：不是拼命，是驱动已跑起来的系统**
