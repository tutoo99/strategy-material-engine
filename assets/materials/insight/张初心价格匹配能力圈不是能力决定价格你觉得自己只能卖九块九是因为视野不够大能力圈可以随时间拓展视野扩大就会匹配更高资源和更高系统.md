## 定价真相

```
你以为：能力决定价格 → 我只能卖9.9
实际：价格匹配能力圈 → 视野扩大就匹配更高系统

从下往上（低手）：先卖9.9 → 试99 → 试999 → 看哪个好
从上往下（高手）：先想好系统 → 价格是最后决策

更深的问题不是"卖多少钱"
而是"这个价格背后服务链条是什么"
"到底能解决多少问题"
```
