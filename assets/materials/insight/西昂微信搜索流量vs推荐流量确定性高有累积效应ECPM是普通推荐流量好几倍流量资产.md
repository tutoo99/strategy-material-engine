## 搜索 vs 推荐

| 维度 | 推荐流量 | 搜索流量 |
|------|---------|---------|
| 用户行为 | 被动看到 | 主动搜索 |
| 稳定性 | 今天爆明天没 | 排名稳定就持续 |
| 确定性 | 低（进池前无收益） | 高（数据可见） |
| ECPM | 低（~十几块/万阅读） | 高（~60元/万阅读） |
| 累积效应 | 无 | 有（排名不掉就持续） |
| 比喻 | 大街上发传单 | 客户主动走进店铺 |

## ECPM高的原因

有明确需求 → 完读率高 → 跳出率低 → 广告充分展现 → ECPM高
