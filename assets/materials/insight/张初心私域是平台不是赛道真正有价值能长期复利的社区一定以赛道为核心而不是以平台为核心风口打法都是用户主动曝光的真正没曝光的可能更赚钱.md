## 核心区分

```
以平台为核心：围绕风口/AI/某平台 → 打法案例都是曝光偏的
以赛道为核心：垂直深耕某个领域 → 长期复利

私域 = 平台（工具/渠道）
赛道 = 你长期深耕的方向（不可替代）

几百个项目跟你关系不大
真正愿意持续做的 = 只跟你自己有关
```
