## 复合思维

```
以前：YouTuber + 程序员 = 两个人
现在：YouTuber + AI编程 = 一个人

三哥的情况：
  测试从业者 + Python/JS/Go/Rust + AI编程能力
  → 技术栈足够
  → 缺的是：找到一个真实痛点 + 动手做出来

大郑的关键：
  不只是会技术，而是"充分理解创作者困境"
  → 先是用户，再是产品经理，最后才是程序员
```
