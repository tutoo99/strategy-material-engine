## 平台创业本质

```
你以为：我在平台上做生意 → 平台应该保护我
真相：每个内容生产者=平台寄生虫 → 平台随时可以卡你

张初心的选择：
6万粉账号被封 → 不抱怨平台 → 不报复举报者 → 感恩有这个机会
→ 在规则下共同生长
```
