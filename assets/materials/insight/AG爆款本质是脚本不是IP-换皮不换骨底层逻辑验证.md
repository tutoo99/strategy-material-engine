# 爆款本质是脚本，不是IP

## 核心发现

AG三个月卡万播期间反复复盘，发现一直模仿皮忘了抄骨。换皮不换骨后：

| IP组合 | 结果 |
|--------|------|
| 富太太vs穷太太（双人） | 数据平平 |
| 穷太太（单人） | 半死不活 |
| **富太太+吉娃娃** | **4000万播放** |

## 跨赛道验证

猫猫故事原版 → KPOP用rumi+蓝猫复刻也爆 → AG用富太太+吉娃娃复刻也爆。

## 结论

脚本内核才是爆款的灵魂，IP只是外衣。一个好的脚本可以换无数种IP反复使用。
