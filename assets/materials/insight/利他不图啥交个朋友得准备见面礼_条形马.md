# 利他：不图啥，交个朋友，得准备见面礼

## 核心观点

亦仁以前说利他，那时我半信半疑。亦仁今年又说利他，今年我坚信来的。为什么又想再多写写了呢？不图啥。交个朋友呗。交朋友，不得准备些见面礼么。

## 认知变化

从"免费给人写多亏啊" → "其实亏不着的" → "不图啥交个朋友" → "见面礼心态"

这个心态转变的核心是：从"我亏了什么"切换到"我收获了什么"。短期看是亏了时间，长期看是交了朋友、积累了人脉、建立了信任。条形马在生财年会上被2017-2019年的读者认出来求合影，就是这些"见面礼"的长期回报。

## 对三哥的启发

三哥做技术号发干货文章，可以参考这个心态——不是每篇都要变现，而是在积累"见面礼"。积累到一定程度，会有人主动找上门来。条形马之前说的"互益交换"也是这个逻辑：先给出价值，价值会以意想不到的方式回来。
