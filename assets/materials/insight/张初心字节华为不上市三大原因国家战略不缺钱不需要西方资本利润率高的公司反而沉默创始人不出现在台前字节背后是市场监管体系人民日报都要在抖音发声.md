## 三大原因

```
1. 国家战略
   股市=美国操控 → 不上市=别人不知道赚多少
   利润率高的反而沉默（网易/腾讯/拼多多创始人不出现在台前）
   → 短视频出现的创始人都不赚钱（至少不是最赚钱的）

2. 不缺钱
   华为=起好名字+站对位置=仗已打完
   字节=抖音干掉所有巨头+人民日报发声=国家宣传窗口
   字节甚至不想让别人知道在干

3. 不需要西方资本
   资本要套利
   中国不是马斯克那套叙事
   高手玩的不是看到的商业模式
```

## 核心判断

```
赚不到钱 ≠ 能力太小
赚不到钱 = 位置太小

没有恒定权力和利益 → 只有动态博弈和合作
```
