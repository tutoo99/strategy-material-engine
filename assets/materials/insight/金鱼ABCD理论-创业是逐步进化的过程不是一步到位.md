# 金鱼ABCD理论

创业不是一步到位，是逐步进化的过程。

## 核心逻辑

做A赚了钱 → 有机会做B → 做B积累了资源可以做C → 做C整合所有资源完成最终形态D

## 经典案例

马化腾：QQ → 微信 → 游戏视频 → 腾讯巨无霸。做QQ时差点50万卖掉，不知道终局。

## 金鱼的五个项目各自教会了什么

| 项目 | 学到的商业常识 |
|------|---------------|
| 母婴纪念品 | 创业应先搞流量，搞到流量再想产品 |
| 小说推文 | 创业必须做调研，不可臆想 |
| 视频号混剪带货 | 要找有红利期的平台和项目 |
| 小红书电商带货 | 要有互联网思维和网感 |
| 视频号直播带货 | 要找正确的对标 |

## 态度

做项目不看喜好度看能不能赚钱，先做了再说。赚不到钱至少提升了能力，为下一个项目做准备。
