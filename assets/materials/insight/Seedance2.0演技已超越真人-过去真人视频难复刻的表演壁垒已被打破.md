# Seedance 2.0：演技超越真人，复刻窗口期

## 核心判断

Seedance 2.0 生成的视频演技已超越真人。复刻真人视频的最大障碍——表演——已被打破。

## 窗口期逻辑

- 模型处于灰度测试阶段
- 服务器承载不了高强度使用
- 国外用户用不了
- = 中国出海圈友独占窗口期

## 当前瓶颈

生成速度慢，15秒视频排队40+分钟，目前无API可调用。
