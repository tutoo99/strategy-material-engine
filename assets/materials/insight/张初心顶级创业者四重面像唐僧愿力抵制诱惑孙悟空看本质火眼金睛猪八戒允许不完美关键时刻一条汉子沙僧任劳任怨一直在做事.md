## 四重面相

```
唐僧（愿力）    → 普度众生 + 抵制诱惑 + 善良帮助他人
孙悟空（本质）  → 火眼金睛 + 看清人妖 + 72变化 + 十万八千里
猪八戒（真实）  → 允许不完美 + 好吃懒做 + 关键时刻一条汉子
沙僧（执行）    → 任劳任怨 + 话少 + 手头一直在做事

四像合一 = 顶级创业者（狙击手）
```
