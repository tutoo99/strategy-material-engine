## 两种生意逻辑

```
热爱的                        能赚钱的
赚不到钱没关系还能坚持        完成一个做另一个
王石做万科（鞠躬尽瘁）         段永平退出做投资
平流层                        大气层
→ 赚到认同但不一定赚到钱       → 可能做成头部但人生更困难

核心判断：
赚不到大钱 = 把体力当唯一筹码
厉害的人 = 人间蒸发偶尔降落
```
