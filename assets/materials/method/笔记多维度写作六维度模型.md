笔记多维度写作六维度模型（郭东超）：

**六维度：** 生活场景 + 人群身份 + 年龄阶段 + 核心痛点 + 情绪记忆 + 行动障碍

每个维度可以组合出大量差异化内容。比如：
- 场景：半夜被梦魇吓醒、考试前焦虑失眠
- 人群：大学生、上班族、宝妈
- 年龄：18-25、25-35、35+
- 痛点：不敢入睡、影响白天工作
- 情绪：恐惧、无助、焦虑
- 障碍：不知道怎么办、试过很多方法没用

**形式：** 纯文字、大字报、长文（5000字也可以）

**流程：** 6维度 + 提示词 + 小红书词包 → 一键生成标题 → 手机手动发笔记

**重要原则：**
- 不用抄别人的标题和封面，别人也是原创也不知道为啥爆
- 可以参考设计UI和逻辑
- 通过优质原创笔记选择优质用户，不能为了流量胡乱做笔记
- 核心要写产品能帮用户解决什么痛点：省钱？省时间？赚钱？某种情绪价值？
