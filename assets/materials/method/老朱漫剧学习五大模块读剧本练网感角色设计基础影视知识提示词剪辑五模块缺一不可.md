## 五大模块

```
1. 读剧本练网感 → 情绪驱动，30秒反转，每集钩子
2. 角色设计 → MJ/即梦/banana，能表达想法就行
3. 影视知识 → 景别+运镜+构图，半天弄明白
4. 提示词 → 让AI跟着你走，先抄后超
5. 剪辑 → 唯一体力活，必须自己练
```

## 避坑

"学AI漫剧不是学写提示词，提示词只是很小一环"
