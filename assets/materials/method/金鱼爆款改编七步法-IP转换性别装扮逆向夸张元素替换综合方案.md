# 爆款改编七步法

保持脚本结构和分镜数量不变，甚至BGM都不变的情况下，对其中的部分分镜进行改编，改编占比30%-50%。

## 七种改编思路

**1. IP转换** — 同脚本直接换成其他IP，比如用火影忍者里面的IP套壳，或者美国山海经套壳

**2. 性别转换** — 角色性别对调，原Mira和Rumi换成Abby和Jinu，原Abby和Jinu换成富家千金和穷女孩

**3. 装扮改变** — 穿华丽的公主裙，或形象物种替换(小猫改小狗改企鹅)

**4. 逆向思维** — 小猫改小狗、感冒改发烧、美人鱼改天使、男孩改女孩、光头改超长头发、胖子改瘦子、老头改年轻人

**5. 夸张化** — 撒钱改撒大量黄金钻石钞票、穷人更穷富豪更富(注意不丑化形象)

**6. 元素替换** — 将部分分镜元素替换成其他爆款元素或对应IP中的元素

**7. 综合改编方案** — 多种改编组合使用，保持总分镜数不变、结构相似

## 铁律：防同质化

有ABCD四个爆款脚本，可以轮流使用微创新，但千万不要AAAA或BBB这么发！大概率逃不过同质化被制裁。爆了之后赶紧联系教练放大。

## 绝对红线

1. 伤害儿童(身体或心理)
2. 血腥恶心引人不适
3. 侮辱丑化IP形象
4. 同质化重复用同一脚本
