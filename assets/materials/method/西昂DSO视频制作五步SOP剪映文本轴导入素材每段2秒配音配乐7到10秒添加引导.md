## 五步SOP

```
1. 剪映创建项目 → 文本轴 + 字体差异化
2. 导入素材 → 多片段，每段2秒，画面要变
3. 配音配乐 → 背景音乐必须有
4. 控制时长 → 7-10秒
5. 添加引导 → 底部引导联系
```

## 两种能拿询单的视频类型

1. 真人讲解型：一人介绍产品+文字（真实感强）
2. 混剪展示型：混剪+价格罗列（信息直观）
