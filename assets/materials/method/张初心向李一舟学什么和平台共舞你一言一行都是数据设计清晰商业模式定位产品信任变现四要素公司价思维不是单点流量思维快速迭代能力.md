## 三学四不学

```
✅ 学什么                    ❌ 不学什么
1. 和平台共舞（你是数据）    1. 泛流量（不是所有行业适合）
2. 清晰商业模式（四要素）    2. 一周一课模型（需要长期主义）
3. 快速迭代（思考时间多）    3. 低宱课程（要ROI不要流量）
                              4. 直播话术（学不会，差距在人设和认知）
```

## 商业模式四要素

```
1. 定位：赚谁的钱？
2. 产品：用什么交付来赚？
3. 持续信任：凭什么持续选你？
4. 变现方式：通过什么销售模型卖出？
```
