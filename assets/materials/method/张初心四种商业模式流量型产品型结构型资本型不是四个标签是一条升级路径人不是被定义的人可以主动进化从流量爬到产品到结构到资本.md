## 四种商业模式

| 模式 | 靠什么 | 本质 | 风险 |
|------|--------|------|------|
| 流量型 | 速度+执行+勤奋 | 一阵风 | 追热点焦虑 |
| 产品型 | 打磨极致 | 把自己当产品 | 慢但稳 |
| 结构型 | 系统思维 | 搭模型看清结构 | 需要认知深度 |
| 资本型 | 资本放大 | 杠杆一切 | 需要前面三层基础 |

## 升级路径

```
流量型（追风）→ 产品型（打磨自己）→ 结构型（搭系统）→ 资本型（放大）

张初心轨迹：
爆款做顺(流量) → 单价30→10万(产品) → 帮人搭系统(结构) → 终点资本型
```

## 核心认知

```
四种模式 ≠ 四个标签
四种模式 = 一条升级路径

最致命的牢笼：内心给自己定的标签
解法：人可以主动进化
```
