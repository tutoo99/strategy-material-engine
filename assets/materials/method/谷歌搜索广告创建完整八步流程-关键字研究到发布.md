# 谷歌搜索广告创建完整八步流程

来自前行「谷歌广告投放保姆级教程（基础篇）」。

## 四层结构理解

```
Ads账号（设置付款/权限/时区/转化追踪）
└── 广告系列（投放目标/类型/每日预算/地区/语言/出价策略）
    └── 广告组（一组主题相近的关键词 + 对应广告文案）
        └── 广告（标题/描述/落地页/附加信息）
```

**预算在广告系列层级控制**，下面的广告组共享这个预算。

## 创建八步

1. **选择目标**：广告系列目标选「网站流量」，转化目标选之前设置好的默认目标
2. **选择类型**：选「搜索」（新手主要投谷歌搜索广告）
3. **设置出价**：前期选「点击次数」，设置每次点击出价上限（参考关键字规划工具数据）；有10-20个转化后再换为「转化次数」
4. **广告系列设置**：
   - 投放网络：**只选谷歌搜索**，关闭合作伙伴（否则预算被泛流量耗光）
   - 地理位置：选目标用户所在地区，前期可选所有国家后续优化；选「所在地」
   - 语言：按网站语言选，多语言各建独立广告系列
5. **AI Max**：前期**不开启**
6. **关键字**：添加之前规划好的关键词，注意匹配类型（推荐词组匹配+完全匹配）
7. **广告文案**：标题+描述+落地页链接，可用AI辅助生成，广告效力达到「良好」即可
8. **每日预算**：根据每次点击费用和目标点击数计算，或直接填可接受的预算

## 关键避坑

- 关闭搜索网络合作伙伴：勾选后广告出现在YouTube等平台，流量不精准
- 出价参考关键字规划工具的最低/最高出价区间
- 前期目标：积累数据让系统学习，不是追求ROI
