## 商业模式前提三问

```
1. 目标是什么？
   → 张初心：和少数人前行，持有顶级公司股份

2. 价值观是什么？
   → ① 筛选为主+培养为辅，为创业者做贡献
   → ② 能力圈之内（知识付费/轻资产/线上为主）
   → ③ 长期主义+极简，少而精
   → ④ 尽可能长期与志同道合的人前行
   → 影响：芒格、段永平、稻盛和夫

3. 热爱是什么？
   → 上课（第一桶金）+ 思考 + 投资 + 观察

关键洞察：
❌ 先看别人赚钱→抄模式（顺序错）
✅ 30万后发现没有任何模式能照抄
✅ 只能融合+创新走出自己道路
```
