## 八步流程

```
1. ChatGPT（思考模式）→ 选题
   等级式结构 / 叙事式结构 → 匹配提示词

2. ChatGPT → 脚本
   ⚠️ 不直接复制！= 当大纲→理解→改→加自己思考

3. ElevenLabs → 英文配音（建议做英文版）

4. GPT/即梦 → 角色参考图
   修改颜色/裤子/上衣不要完全一样

5. ChatGPT → 脚本转图片提示词（分段生成）

6. 即梦 → 剧情配图
   上传角色参考图保持一致 16:9

7. 剪映 → 剪辑
   关键帧+转场+特效 1080P
   油管不加背景音乐

8. ChatGPT+OpenArt → 封面
   两种风格：第一视角 / 白底+人物
```
