# 形式类爆款七大元素创作法

来源：露酱(YouTube深海圈带货板块教练)+曹教练总结

## 核心逻辑

形式爆款拿流量的底层逻辑：乱花渐欲迷人眼，用套路、画面、音乐等元素对用户进行感官冲击，让用户放弃思考、不知不觉间就把视频看了一遍又一遍。

## 七大元素

### 1. IP
和脚本爆款一样，各种IP都可以做形式爆款，IP就是个自带受众流量的壳。热门IP参考：brainrot意大利脑腐、kpop、罗布乐思。

### 2. 视觉猎奇
变身、跳舞、走秀、跳水、跳床、跳岩浆、跳飞机。从视觉奇观出发，大幅度动态、高难度甚至不可能的动作，人类不常拍或拍不了的东西。

### 3. 感官解压
切蛋糕、吃东西、砸东西、泼液体、手办。从感官刺激出发，带有解压属性，看着很爽，结合声音吸引力更强。

### 4. 非常规视角
微缩场景、监控器视角、第一人称视角。从视角出发，不常见、新奇的视角。

### 5. 极端角色
巨人、巨物、机器人、怪人。从人物出发，极端、猎奇的人物或场景特征。

### 6. 视觉信息量
四宫格、多宫格、首尾循环、转绘、分屏。增加相同时间内的视觉信息量，或误导用户增加观看时间。

### 7. 人性欲望
美女、怀孕、钱、帅哥、明星、擦边。利用贪、色等底层欲望吸引眼球。

## 使用建议
- 元素组合几乎无穷尽，不要被对标限制思考框架
- 兼职做或脑洞大喜欢搞抽象的人适合这个赛道
- 有圈友靠形式爆款做到几千刀/月
