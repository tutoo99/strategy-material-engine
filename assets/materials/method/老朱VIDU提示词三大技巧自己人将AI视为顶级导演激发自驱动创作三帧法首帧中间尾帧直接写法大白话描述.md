## 三大技巧

### 自己人
```
把AI当顶级导演/武术指导
→ 赋予身份+宏大目标
→ 激发自驱动创作
→ 不机械执行琐碎指令
```

### 三帧法
```
首帧状态 + 中间过程 + 尾帧结果
解决动作逻辑混乱
```

### 直接写法
```
大白话描述画面
不写运镜、不写景别
让AI自己发挥
```

## 关键原则

参考生不要过度使用专业名词，释放AI能力
