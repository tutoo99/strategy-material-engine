## 群响优化六步

```
1. 被动收入思维
   → 怎么离开公司还能运转
   → 不是持续在线卖体力

2. 创业者真正需要的不是信心
   → 赚3000=打赏=卖体力
   → 没有心力能量创业不成功

3. 创造标准化东西
   → 从所有项目找3-5个深耕
   → 要么拿大结果（3000-5000万）
   → 要么总结方法论成为训练产品

4. 照顾好私董会
   → 深刻解决需求，组队帮解决

5. 生存层用户（几十万几百万）
   → 需要清晰定位和商业模型
   → 精准指导快速复制裂变
   → 不是分享案例（信息太多不知道怎么办）
   → 意见领袖告诉他怎么做=层次不同

6. flomo模式
   → 知识管理工具卖会员
   → 不知创始人但会员到处都是
   → 被动收入一年上千万不用在台前

关键：下个月没工资=一年前方向就不对
```
