# YouTube找对标最土方法(新手通用)

## 步骤
1. 新注册YouTube账号
2. 搜索教练推荐的赛道账号，关注
3. 从视频背景音乐找到n多个视频，刷、看、点赞
4. 关注更多在更新且符合赛道的账号
5. 刷新首页 → 算法推送无穷赛道内容

## 印度赛道判断
- 优势：体量极大，一周半数千万爆款的号很多
- 劣势：单价最低，同样千万播放赚最少
- 策略：先起号开通YPP，再切换赛道或继续深耕

## 辅助工具
盖洛普评测报告 + GPT分析项目匹配度(查理得出五星匹配)
