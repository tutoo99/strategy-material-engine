# 爆款脚本收集四标准

## 来源策略

跨IP + 跨平台：印度、真人、影视剧切片、生活趣事。国内视频改造后发油管照样有量。

## 四大筛选标准

| # | 标准 | 说明 |
|---|------|------|
| 1 | 播放量高 | 被市场验证：油管千万/小红书10万/抖音几十万 |
| 2 | 便于AI复刻 | 排除心理活动（眨眼/暗示），选外显行为 |
| 3 | 情感曲线 | 温馨感人/悲伤凄惨/搞笑反转/抽象 |
| 4 | 人性主题 | 人性抉择/贫富差距/猎奇/贪婪 |

## 国内→油管验证

小红书10万 + 抖音59万的视频 → 改编后油管3200万播放
