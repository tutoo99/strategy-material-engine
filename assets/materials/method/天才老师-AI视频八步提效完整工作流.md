# AI视频八步提效工作流

1. **找选题** → 系统前置搜集整理，从预处理材料做判断
2. **拆解原视频** → 钩子/情绪/节奏/密度/留人段位/心理机制
3. **改编** → 不是复制而是翻译，输出改编方案
4. **补提示词和素材** → 每段画面、提示词类型、参考图、必保留镜头
5. **提交生成** → 提交前检查+提交后确认+失败补救纳入同一流程
6. **筛选整理** → 哪版好/哪段可用/哪要重来/哪值得保留
7. **归档** → 拆解+建议+提示词+素材+结果+最终版收回到固定结构
8. **结果回流** → 数据变成下一轮优化输入

核心：不是把某一步做快，而是把整条链路接顺。
