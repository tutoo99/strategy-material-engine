## 操作流程

```
1. ad.qq.com/AiDSO查关键词 → 找黄金关键词
2. 微信搜一搜验词 → 看现有内容竞争
3. 竞争少/旧文章多 → 机会大
4. 创作高质量原创内容 → 信息增量
5. 优化标题和内容结构
6. 持续监测排名和流量
```

## 黄金关键词

有搜索量 + 竞争低 + 长尾词
