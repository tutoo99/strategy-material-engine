# 内容提效五步上手方法

1. **找最重复动作** — 别自动化全部，先从一个点切进去
2. **任务标准化** — 固定输入→固定输出→固定检查
3. **归档当生产一部分** — 今天多花5分钟整理后面省50分钟重找
4. **让AI帮你拆问题** — 不只问怎么做，持续追问拆开问题
5. **先建最小闭环再扩** — 选题→拆解→改编→归档，跑顺再接生成/发布/数据回流
