## 五环模型

```
        梦想蓝图
       /          \
    价值观          资源
       \          /
    经历 —— 基因

五环交汇点 = 你的唯一护城河
```

## 五环详解

| 环 | 问题 | 本质 |
|----|------|------|
| 基因 | 你天生擅长什么？ | 做起来轻松别人吃力 |
| 经历 | 过去10年踩了什么坑？ | 不总结是消耗，总结了是资产 |
| 价值观 | 不为钱你也愿意做什么？ | 骨子里的热爱，撑过孤渡期 |
| 梦想蓝图 | 5-10年后你想成为什么？ | 设定目的地决定今天敢做难事 |
| 资源 | 你手上真正有什么？ | 不需要变成别人，打好手上的牌 |

## 核心公式

```
AI时代 → 信息切到原子级 → 定位维度必须≥5
五环全部叠在一起 → 唯一交汇点 → 只有你站在那里
→ 别人抢不走、复制不了、卷不掉
→ 方向、定价、路径自动浮现
```
