## 五大筛选法则

```
显性素质
├── 1. 目标感：把事情做成的决心，死也要拿到结果
└── 2. 网感：亲手剪辑300条+，会聊天

隐形素质
├── 3. 长期主义：愿意花3-5年，不着急变现
└── 4. 人品：当面骂可以，背后说坏话永不录用

加分项
└── 5. 稳定亲密关系：下半身不稳上半身无法修炼
```

## 面试方式

不要看案例，只要描述——哪些事情证明你有这五个特质
