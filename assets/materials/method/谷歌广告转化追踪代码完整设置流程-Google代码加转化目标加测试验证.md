# 谷歌广告转化追踪代码完整设置流程

来自前行「谷歌广告投放保姆级教程（基础篇）」。

## 为什么必须配好转化追踪

投广告前期最大目的是积累数据，让谷歌了解你的网站需要什么样的用户、什么样的用户会付费。没有转化追踪，广告系统就是瞎子。

## 设置步骤

### 1. 添加Google代码
- Ads后台 → 工具 → 数据管理器 → Google代码 → 管理
- 获取AW开头的ID，选手动添加代码，复制给AI编程工具（如Codex）添加到网站
- 如果网站已有GA4代码，只需让AI稍微改动即可
- 添加完后测试，结果出现✅才算成功
- 建议把Google代码名称改成网站域名方便识别

### 2. 创建转化目标
- 目标 → 摘要 → 新建转化操作 → 网站上发生的转化
- 数据源选Google代码（比GA4更准确）
- 转化类型选「购买或订阅」（最终目的是用户付费）

### 3. 转化目标设置
- 操作优化：**主要操作**（用于智能出价优化）
- 价值：选「为每次转化使用不同价值」，填兜底值
- 统计方式：选「每一次」
- 增强型转化：确保勾选

### 4. 添加追踪代码
- 选Google代码方式 → 查看事件代码段 → 复制给AI编程工具添加
- 核对代码段里`send to`后面的ID是否正确

### 5. 测试验证
- 用Chrome浏览器（不要用AI浏览器，可能被拦截）
- 支持本地测试，可先本地验证再部署上线
- 触发对应转化行为（如真的完成一次沙盒支付）来验证

### 6. 补充次要转化目标
- 重复上述步骤，但操作优化选**次要操作**，统计方式选**一次**
- 推荐组合：购买（主要）、试用（次要）、注册（次要）、加入购物车（次要）
