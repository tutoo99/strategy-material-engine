# 脚本改编四层法

## 核心前提

曹教练：提炼出脚本内核 + 成功做出爆款 = 无限改编能力

## 四层改编

### 1. 角色互换
rumi偷鸡 → jinu偷鸡，换IP角色互换。

### 2. 主题改编
偷鸡 → 偷衣服/偷手机/偷首饰/偷泡面 = "偷宇宙"系列

### 3. 结尾改编
合家欢 → 悲剧，主角即将成功时被车撞/被广告牌砸/被地铁撞/被警察抓。主角抱遗物哭泣。

### 4. 抽象改编（最高级）
抽离内核：一个人为隐藏本来面目进行改造。
变体：男变女、女变男、丑变美、机器人变真人、老变少。
爆点：贫困偷盗、顶替荣华、做好事不留名。
拆穿：无意举动、习惯改变、记忆不能共享。

## 实战数据

单个偷盗脚本改编多版本 = 1.5亿播放
