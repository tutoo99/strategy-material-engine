## 三法则

```
1. 不做流量型 → 做信任型（共生）
   给用户真实感 + 给平台优质内容耐心
   ≠ 用投放掠夺客户

2. 长出自己造血能力
   假如没流量了 → 私域/其他平台怎么活？
   没有造血能力 = 血液别人提供 = 随时断供

3. 信任潜质 → 转化率自然高
   梳理好定位 → 第一个粉丝当天付费
```

## 两种生意

```
掠夺型：投放抢客户 → 和平台对抗 → 被卡就死
共生型：优质内容+真实感 → 和平台共生 → 脱离平台也能活
```
