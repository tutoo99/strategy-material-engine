## 七步操作

### 第1步：无痕模式探索
打开浏览器无痕模式，不登录账号，在搜索框搜一个感兴趣领域的视频开始看，让首页推送相关内容。

### 第2步：收集关键词
刷YouTube首页到底部，收集各种细分类型视频的关键词。

### 第3步：操控信息茧房
用收集到的关键词搜索视频，逐个点开播放，让算法知道你喜欢这类内容。

### 第4步：二刷首页找闷声账号
回到首页刷内容，按筛选标准寻找符合条件的小而美账号。

### 第5步：搜索栏精确筛选
搜索细分关键词，选择本月+20分钟以上+观看次数排序，找万播以上的作品。

### 第6步：验证竞争环境
把对标频道最热门视频标题拿去搜，看竞争对手是否大于5个，不符合标准就换赛道或放宽标准。

### 第7步：火速开测
符合标准立刻开几个账号同时测试，慢一步就有模仿者。

## 筛选标准（铁律）

| 指标 | 标准 |
|------|------|
| 竞争者数 | 小于5个 |
| 赛道内视频数 | 小于50个 |
| 对手发布时间 | 不超过3个月 |
| 异常值播放 | 足够多 |
| 对手订阅数 | 3万以下 |
| 播放vs订阅 | 平均播放远大于订阅数 |

## 核心认知

- 长视频赛道藏在夹缝里，短视频赛道摆在明面上
- 赛道就像被低估的股票，能公开拿出来卖的赛道等于没有护城河
- 找到好赛道请紧闭你的嘴，或者只分享给绝对信任不会无脑抄袭的人
- 练出选赛道的眼光比磨出图出视频细节重要百倍
