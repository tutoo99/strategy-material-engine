# AI生图敏感词替换表

## 核心原则

不带感情色彩，只描述状态。

## 替换对照

| 敏感词 | 替换为 |
|--------|--------|
| 血液 | 红色液体、红色污渍 |
| 伤口 | 裂口 |
| 窒息 | 双手捂着脖子，脸部轻微泛红 |
| 惊恐 | 眼睛瞪大、嘴巴张大 |
| 怒骂 | 眉头紧皱地交谈 |
| 嘲讽 | 眉毛上挑地大笑 |
| 皮肤细节 | 肉粉色带纹理平面 |
| 身体内部 | 粉红色带褶皱管道 |

## 工具栈

- 生图：Nano banana pro + will大佬批量工具
- 视频：grok（便宜大碗）、hailuo（打斗/变身）、flow（音画直出）
- 配音：veo3直出 / elevenlabs
- 音效：剪映自带
