## 性格 × 商业模式

```
外向型人
→ 别硬逼当工匠
→ 撮合资源 / 做平台 / 做销售

内向型人
→ 别做销售别把自己耗死
→ 沉心做产品 = 你的主场
```

## 60分到90分

```
每项能力可能都是60分
→ 选定一项加大筹码+时间投入
→ 做到90分
→ 这一项 = 你的护城河
```
