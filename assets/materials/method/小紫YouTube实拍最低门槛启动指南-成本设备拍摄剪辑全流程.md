# 实拍最低门槛启动指南

## 成本清单

| 项目 | 成本 |
|------|------|
| KPOP衣服 | ~80元/套 |
| 假发 | ~60元 |
| 其他道具 | 月一两百元 |
| 手机支架 | 15元 |
| 灯泡（夜拍） | 50瓦 |
| **合计启动** | **~500元内** |

## 设备真相

苹果XR拍的视频照样过2000万播放。不用相机、不用稳定器、不用租场地。

## 拍摄核心

- 四种景别来回切换：近景、中景、远景、特写
- 运镜：推、拉、摇、移、环绕
- 两部手机：一部放对标视频，一部拍，一帧一帧模仿
- 冷白灯光 > 暖黄灯光

## 演员选择

**理解能力到位 + 认知水平高** = 效率高。小紫和父母拍时，一个表情拍了40分钟（一遍能过的东西）。建议找认知高的演员或专业人员。

## 模仿精髓

模仿的是精髓不是外表。要丑就丑得出奇，要恶心就足够恶心。
