# YouTube Shorts六步变现路径

## 四种变现方式

1. **平台广告分成**：YPP广告分成，收益=互动播放量(万)×RPM×10
2. **联盟营销**：视频中出现商品链接，观众下单可获佣金（不限于链接中的商品）
3. **知识付费**：提示词、小插件、工作流、爆款知识库等产品化变现
4. **商单合作**：MCN/品牌方主动寻求合作

## RPM参考

- Shorts无语言：0.01~0.06
- Shorts语言类：0.05~0.3
- 长视频一般：3左右
- 长视频金融财经：几十甚至上百

## YPP开通条件

- 初级YPP：1000订阅 + 过去365天4000小时观看时长
- 或：过去90天1000万Shorts播放

## 变现路径

1. 学会AI基本操作 → 2. 开通YPP → 3. 深耕赛道突破卡万播 → 4. 拓展多条变现渠道
