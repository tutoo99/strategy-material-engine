# YouTube shorts爆款公式=爆款IP x 爆款脚本

## 公式本质
爆款视频 = 爆款形式 x 爆款脚本。爆款IP只是爆款形式的一种，爆款形式还包含好看的形象、可爱治愈的画风等。

## 爆款IP七大类
1. 卡通动物：猫猫狗狗
2. 影视IP：钢铁侠蜘蛛侠等漫威形象
3. 动漫IP：七龙珠宫崎骏系列
4. 体育明星：梅西C罗内马尔(YouTube流量超级大)
5. 政治名人
6. 地方特色：如印度脸，印度人就爱看
7. 网络流行Meme：labubu、美国山海经、kpop猎魔女团、星期三、鱿鱼游戏等

注意：IP有周期性，不是所有IP一直有流量。AI山海经2024年4-7月火爆，现在流量不好。蜘蛛侠流量一直稳定。

## 爆款脚本的验证
同一脚本时隔一年复刻还能再爆：
- 2025.5.29：3亿播放
- 2025.6.6：3900万播放
- 2026.2.24：4000万播放

说明触动情绪的好故事是长青的，靠内容吸引人不靠形式洗脑。

## 围绕IP做内容三层递进
1. 简单套用IP角色形象，只借用外壳演绎脚本
2. 轻度二创，深度将IP与脚本内容结合
3. 完全围绕IP演绎，深度参考IP背景制作高度契合的内容

大部分AI创作者做到第一层和第二层就好。

## 关键提醒
AI模型能力已经强大，YouTube不再允许SOP红利让很多人享受。穿越周期靠基本功扎实+心力愿力强大，不是等着喂饭。
