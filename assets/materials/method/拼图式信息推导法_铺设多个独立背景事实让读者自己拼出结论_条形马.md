# 拼图式信息推导法

## 方法描述

不直接告诉读者答案，而是先铺设多个独立的背景事实（拼图），让读者自己拼出结论。

## 条形马的示范

他铺设了4块拼图：
1. 抖音变现通路已通（星图按播放量计费）
2. 鸡汤粉不值钱但鸡汤内容跨平台易爆
3. 不露脸也能做鸡汤视频（电影片段+念白 / 金句截图+配乐）
4. 抖音小程序支持虚拟付费且可Track分发来源

读者自己拼出：在抖音做心理测试小程序，通过分佣让鸡汤号带货。

## 为什么有效

- 读者参与了推导过程，结论变成了他自己的（自我说服）
- 每个拼图单独看都平淡无奇，降低了防备心
- 组合的瞬间产生"原来如此"的顿悟感，比直接说结论更有冲击力
- 比直接说"做心理测试小程序赚钱"有说服力得多

## 适用场景

- 写公众号文章引入新观点时
- 做项目推荐或付费社群分享时
- 任何需要说服他人的商业论述
