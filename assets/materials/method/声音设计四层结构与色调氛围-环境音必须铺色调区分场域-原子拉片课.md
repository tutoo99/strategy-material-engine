# 声音设计四层结构与色调氛围

## 声音四层结构

原子把声音设计分为四层，从底到顶：

```
┌──────────────────────────────┐
│     第4层：配乐（情绪强化）    │  ← 铺垫情绪走向
├──────────────────────────────┤
│     第3层：音效踩点（节奏）    │  ← 刀落地/脚步/关门
├──────────────────────────────┤
│     第2层：人声/台词          │  ← 要听清
├──────────────────────────────┤
│     第1层：环境音（地板）      │  ← 必须铺！
└──────────────────────────────┘
```

### 第1层：环境音（最关键也最易被忽略）

原子原话：环境音必须铺，不铺环境音的画面就是假的。

环境音是什么：风声、雨声、城市嘈杂、室内空调声、远处人声等。这些声音让观众觉得"这个场景是真实存在的"。

新手最容易犯的错：画面做得很精致，但声音是"真空"的——只有台词和配乐，没有环境音。一看就知道是AI做的。

### 第2层：人声/台词

精品内容建议用真人配音。AI配音目前在情绪表达上还不够自然，尤其是需要细微情绪变化的场景。

### 第3层：音效踩点

音效配合画面节奏是快节奏片子的骨架。刀落地的"咣"、关门声、脚步声——这些声音点踩在画面切换的节拍上，让观众"听"到节奏。

### 第4层：配乐

配乐不是铺满全片，而是有起有伏：
- 情绪铺垫时：低沉、缓慢
- 高潮时：激烈、快节奏
- 危险暗示时：低频嗡鸣、不和谐音

## 声音铺垫角色（高阶技巧）

怪物出现前，先铺低频音效。观众还没看到怪物，但已经"感觉到了危险"。

拉片《我的治愈系游戏》中：男主说"这房子不安全"→闪回→回到主观事实接全景→接一个刀的特写——刀出现时配合了音效变化，让观众从"叙事"状态被拉到"紧张"状态。

## 色调=最低成本的氛围工具

原子在拉片中特别强调色调的分析：

| 色调 | 氛围 | 适用场景 |
|------|------|---------|
| 蓝绿冷调 | 悬疑、日常、疏离 | 动态漫最常用色调 |
| 黑红 | 压抑、诡异、嗜血 | 恐怖/暗黑题材 |
| 暖黄 | 温馨、怀旧、安全 | 回忆/治愈场景 |
| 灰白 | 空洞、死亡、冰冷 | 特殊情绪场景 |

**为什么动态漫蓝绿调多？** 因为安全、出片稳定、AI生成效果好。

**黑红慎用**：不是亮红，而是黑红。亮红会显得廉价，黑红才有压迫感。

原子原话："色调是区分场域、事件、环境、情绪，非常直接直观并且很省事的一种方式。天生对色调有感知的人，非常适合走这一行。"
