# AI短剧提示词二次定制模板

## 为什么需要二次定制
AI分镜师/图像设计师输出的原始提示词不一定符合实际需求。雨文结合做AI短视频的经验，将提示词二次转化为更贴合使用习惯的格式。

## 三合一模板

### 图片提示词格式
```
[主体]
角色：角色A
表情：开心
动作：角色A坐在桌前，双手放在桌上。
[环境]一个现代风格的厨房，背景是橱柜和灶台。
[时间]白天
[天气]下雨
[视角]平视
[景别]中景
```

### 视频提示词格式
通用结构 = 场景激活三要素：主体动态化+环境氛围化+镜头电影化

```
[核心画面描述] 一节空荡荡的现代地铁车厢。
[主体动作/表情的细微变化] 乘客们随列车行驶轻微晃动。
[环境元素的动态效果] 窗外隧道灯光快速掠过。
[镜头运动方式] 镜头缓慢向前推进（Slow Push In）。
[风格与画质] 电影感，高对比度，4k画质。
视频画面连贯，流畅，符合现实运动规则，不要出现其他角色。
```

### CSV输出
三种信息合并为一张CSV表格，每列：图片提示词 | 视频提示词 | 对白
格式要求：每个单元格用英文双引号包裹，防止内容中逗号导致错位。
