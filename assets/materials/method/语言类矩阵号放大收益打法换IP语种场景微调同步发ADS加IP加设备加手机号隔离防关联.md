# 矩阵号放大收益打法

## 核心策略

直接换IP、语种、场景形式微调同步发矩阵号，去放大脚本和即梦的制作红利。

## 防关联

矩阵号建议：**ADS + IP + 设备 + 手机号 最细粒度级别隔离。防止关联。**

## 微调原则

- **同IP复刻**：微调幅度要大，场景和恶搞形式大改
- **不同IP**：因为受众不同，微调幅度可小一些
