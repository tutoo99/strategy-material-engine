# 即梦视频提示词稳定核心三公式

## 公式：参考图精准指定 + 分镜秒级控制 + 特写镜头指定

1. **参考图指定**：清晰使用@符号绑定角色与场景图、每个分镜首帧图，减少AI随机发挥
2. **分镜秒级控制**：在提示词中明确描述每个镜头时长（如"0-2秒画面，xxx"），精准控制节奏，减少后期剪辑
3. **特写镜头指定**：需要突出情绪或细节时明确写出"特写镜头"，引导AI聚焦

## 关键注意事项

- **按场景隔离提交**：同一任务只含同一场景内剧情，不同场景分批提交
- **精简入参**：只上传本次生成必须用到的角色图和场景参考图
- **禁止字幕**：语言类视频加限定词"视频不要出现任何字幕"
- **中近景优先**：shorts以中近景为主，远景容易糊脸

## 提示词示例

```
根据三人的动作和对话，自由运镜和切镜，中近镜特写。（视频不要出现任何字幕）。
0-2秒：双马尾红发女人虚弱地微睁开眼，虚弱的说："you all need take care..."，随后缓缓闭上双眼，手臂彻底失去力量无力地滑落向地面（特写镜头）。
3-5秒：双丸子头黑发女人崩溃地大哭，将红发女人紧紧搂进怀里。麻花辫紫发女人仰头向着刺眼的阳光绝望地大喊，"Mira! No! Don't leave us!"
```
