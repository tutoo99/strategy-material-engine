## 0代码用AI开发工具的方法

濯生是0代码基础的实体行业创业者，在YouTube Shorts制作过程中被"逼出"了一个Python工具（AI Shorts Studio v1.5）。

### 关键原则
- **不解决所有问题，只解决当下最确定、最浪费注意力的那一段**
- **定位明确**：验证流程的工具，不是对外售卖的产品
- 只要工具可以稳定跑通流程、减少切换次数、让注意力回到脚本和判断上，就足够了

### 开发流程
1. **梳理**：把整个生产流程从头到尾梳理一遍
2. **标注**：标出哪些步骤是高频重复的、标准化程度最高的、不需要太多创意判断的
3. **确认**：真正消耗精力的不是创作本身，而是机械的处理、切换和等待
4. **开发**：用Claude Code+Gemini/GPT完成需求，Gemini写指令给CC，中间问题交叉验证
5. **重构**：用Claude Code重读航海手册重构架构，业务端代码可配N8N工作流

### 产出
AI Shorts Studio v1.5，完全免费，把Shorts生产流程提效，从4小时压到1小时。
