# 镜头衔接铁律与构图角度

## 四条铁律

行业内通用的镜头衔接规则：

| 规则 | 含义 | 示例 |
|------|------|------|
| 动接动 | 运动镜头接运动镜头 | 推镜头→摇镜头 |
| 静接静 | 固定镜头接固定镜头 | 中景固定→近景固定 |
| 大接大 | 大景别接大景别 | 全景→中景 |
| 小接小 | 小景别接小景别 | 特写→近景 |

**特殊例外**：刻意打破可以制造戏剧效果。例如运动镜头突然接静止镜头 = 情绪突然定住的张力。

**关键细节**：运动镜头的落幅（运动结束后的静止画面）可以接静态镜头。推镜头推到尽头静止了 = 此刻是"静"，可以合法接下一个固定镜头。

## 硬切的真正定义

原子纠正常见误区：**硬切不是"景别跳跃"，而是"镜头之间没有逻辑关系"。**

拉片实例：手拿酒杯（特）→手喝酒（中近），景别跳了一级，但不构成硬切，因为：
1. 因果关系：拿酒杯→喝酒，逻辑通顺
2. 动作方向一致：手都是向上的动作
3. 主体一致：同一个人、同一个动作链

## 180度轴线

摄影机只能在被摄体的180度范围内移动，不能"越轴"（绕到对面去拍）。

越轴的后果：人物运动方向混乱——越轴前人从左侧入画，越轴后可能从右侧入画，观众会觉得人物在"转圈圈"。

AI漫剧制作中尤其要注意：AI生成的分镜经常不注意轴线一致性，人工分镜的优势就在这里。

## 构图九种

| 构图 | 效果 |
|------|------|
| 三分法 | 最安全，适合对话场景 |
| 对称构图 | 庄重、仪式感 |
| 引导线构图 | 引导视线到主体 |
| 框架构图 | 聚焦、窥视感 |
| 对角线构图 | 动感、张力 |
| 中心构图 | 突出、强调 |
| 留白构图 | 孤独、意境 |
| 倾斜构图 | 不安、失衡（**慎用**） |
| 前景遮挡 | 层次感、窥视 |

**倾斜构图慎用**：原子在拉片中明确指出，AI做的倾斜构图经常水平线都不直，透视也有问题，效果"很丑"。除非你确定AI能做好，否则不要轻易用。

## 角度=情绪

| 角度 | 情绪效果 | 适用场景 |
|------|---------|---------|
| 平拍 | 客观、中性 | 大部分叙事 |
| 仰拍 | 恐慌/弱势/崇拜 | 角色害怕时 |
| 俯拍 | 上帝视角/强势/冷漠 | 看扁对方、全景交代 |
| 侧拍 | 偷窥/疏离 | 隐秘观察 |

**重要提醒**：仰拍不只是"崇拜"，也可以表达"恐慌/害怕"。选择角度前先确定角色此刻的情绪，再匹配角度，而不是根据物理位置选角度。

拉片实例：怪物面对男主时没用俯拍。按逻辑怪物应该"俯瞰"男主，但导演选了仰拍——因为此刻要表现的是**男主的恐慌**，不是怪物的强势。
