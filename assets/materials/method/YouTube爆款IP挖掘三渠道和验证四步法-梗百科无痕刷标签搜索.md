# YouTube爆款IP挖掘三渠道和验证四步法

## 渠道一：梗百科
- 关注抖音或B站梗指南账号(如B站space.bilibili.com/1544008396)
- 定期刷快速补齐网络热梗
- 拿IP名称到YouTube搜，看有没有千万以上甚至上亿播放视频

## 渠道二：无痕刷YouTube首页
1. 开无痕模式进入YouTube
2. 先点shorts，等右侧短视频完全加载
3. 再点首页，不断刷新
4. 看到没见过的形象→截图→问AI或Google以图搜图→拿名称去YouTube搜

## 渠道三：全球影视游戏资讯
关注近期热门电影、动画、游戏，经典案例：
- Squid Game(鱿鱼游戏)
- Demon Slayer(鬼灭之刃)
- Minecraft(我的世界)
- Zootopia(疯狂动物城)

## 验证方法
1. YouTube搜索关键词
2. 过滤：热门程度 → 本月 → 短视频
3. 看近几周有没有大几百万甚至千万播放
4. 反例：英雄联盟印象流量大，实际本月连一条500万播放都没有

## 进阶：标签搜索法
点进视频→三个点→说明→蓝色标签→进入标签页看所有带该标签的视频。绕过标题关键词搜索的局限。
