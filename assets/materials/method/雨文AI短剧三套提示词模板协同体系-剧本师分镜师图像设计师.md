# AI短剧三套提示词模板协同体系

## 整体架构
三套提示词形成流水线：剧本创作师输出剧本 → 分镜师输出分镜表 → 图像设计师输出关键帧提示词。每步之间通过JSON状态摘要传递上下文。

## 剧本创作师 v1.1
- 4步流程：创意定型 → 全剧设计 → 剧本生成 → AI交付
- 输出：多集大纲、角色设定、逐集标准剧本（90-120秒/集）、场景清单、角色清单
- 质量：开头15秒有钩子，结尾有卡点，镜头标注清晰，对白自然简短

## 分镜师 v3.2
- 标准模式4步：剧本解析 → 画面设计 → 技术优化 → 质检交付
- 快速模式：详细剧本（含镜头标记）一步直出
- 10秒规则：AI视频生成限制，超过10秒必须拆分
- 镜头编号体系：S01-01（集号-镜头号）

## 图像设计师 v2.0
- 阶段1：人物参考图公式 = {人物名},{年龄}{性别},{发型发色},{脸型},{体型},{服装},{配饰},{特征},全身正面白色背景,二次元动漫风格
- 阶段1：场景参考图公式 = {场景名},{主要元素},{环境细节},{色调},{光线},空镜无人物,二次元动漫风格
- 阶段2关键帧公式 = {动作姿势},{表情情绪},{场景环境},{光线氛围},{镜头构图},二次元动漫风格
- 核心规则：有参考图时绝对不重复描述人物外貌（发型/脸型/体型/服装），这些在参考图里

## 关键帧拆分规则
- 空镜/建立镜头：1帧
- 纯对话无动作：1帧
- 静态特写：1帧
- 明显连续动作：3帧
- 运镜跟随/摇摄：2帧
- 多人场景：看最复杂的那个人
