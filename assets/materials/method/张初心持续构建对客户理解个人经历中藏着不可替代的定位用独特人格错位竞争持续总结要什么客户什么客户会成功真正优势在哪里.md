## 客单价进化背后的动作

```
30块 → 10万 → 30万 → 300万
↑        ↑       ↑
不停在回答四个问题：
1. 我要什么样的客户？
2. 什么样的客户会成功？
3. 我站在哪个位置？
4. 我真正的优势是什么？
→ 停止积累 = 停止涨价
```

## 正反模式

```
正确（深耕）：找到终身客户 → 积累判断力 → 客单价持续涨
错误（割一把）：看哪个火做哪个 → 一拥而上 → 今年赚明年亏 → 挺不过三个月
```
