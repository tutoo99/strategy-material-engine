## 获客系统

```
普通人的内容流程（累）：
写文案 → 拍摄 → 剪辑 → 发布 → 试错

曲曲的内容流程（省）：
解决问题（直播连麦）→ 自然生产内容 → 切片分发 → 放大传播
```

## 直播连麦三位一体

| 功能 | 说明 |
|------|------|
| 获客 | 公域直播吸引新流量 |
| 转化 | 连麦中直接展示方法论效果 |
| 内容源头 | 切片分发到各平台持续引流 |
