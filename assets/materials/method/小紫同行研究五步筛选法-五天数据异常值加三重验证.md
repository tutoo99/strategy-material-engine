# 同行研究五步筛选法

## 五步流程

1. **找10个同行账号**，最近五天的数据全部看一遍
2. **筛选异常值**：五天内过500万/1000万/2000万的视频 = 好脚本
3. **验证黄金三秒**：筛选出的视频是否在前三秒抓住了用户
4. **验证历史数据**：该题材以前用过没？以前数据怎么样？如果以前也好，这个素材绝对没错
5. **融入五官刺激+爆款元素**：僵尸、长指甲、砸玻璃等

## 核心心法

- 掌握这个能力前花了 **不下100小时** 沉浸在油管里
- 先有根基，才有模仿超越
- 抄别人的不如抄自己的：趁同行还没发现之前，自己已经火过一遍，再抄一遍自己
- 同行10条/20条视频里穿插一条以前发过的，照样火
