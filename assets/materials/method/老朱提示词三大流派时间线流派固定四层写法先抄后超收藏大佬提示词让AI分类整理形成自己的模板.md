## 三大流派

### 时间线流派
```
[时间区间] + [画面内容] + [运镜/景别/氛围]
0-3s：全景，固定镜头，男主站在雨夜巷口...
3-7s：中景，推镜头，男主缓缓抬起头...
```

### 固定四层
```
1. 风格氛围层：质感约束+风格词
2. 运镜景别层：运镜+景别+视角
3. 核心画面层：大白话描述剧情
4. 细节强化层：特效+质感补充
```

### 先抄后超
收藏 → 整理 → AI分类 → 模仿 → 形成自己模板
