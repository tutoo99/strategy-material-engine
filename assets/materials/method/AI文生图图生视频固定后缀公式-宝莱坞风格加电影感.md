# AI文生图图生视频固定后缀公式

## 文生图后缀
```
宝莱坞电影风格，电影感光线，高饱和度，高清细节
```

## 图生视频后缀
```
画面主体动态呈现，人物面部清晰，动作自然流畅，环境光，符合逻辑，极致细节，超真实动态捕捉。人物脸部不变形。清晰自然，不模糊，高质量，没有瑕疵
```

## 保持一致性
- 先定义固定人物形象描述
- 每个分镜都包含固定形象描述
- 不一致时加参考图或反复抽卡

## 关键避坑
动作描述必须和图片实际情况完全一致。
