## 六步SOP

```
1. 剧本 → AI写大纲+拆单集（30集/2万字）
2. 资产设计 → 角色三视图 + 场景(21:9)
3. 拆分镜 → AI拆（新手可省）
4. 提示词 → 批量提交即梦（每条提交两遍）
5. 剪辑 → 先铺音轨再卡画面
6. 补镜头 → 循环直到成片
```

## 团队配置（精品）

| 角色 | 职责 |
|------|------|
| 导演 | 剧本、分镜、提示词、整体方向 |
| 制作 | AI生成、筛选素材、粗剪 |
| 剪辑 | 精剪、配音配乐、成片 |
