# 爆款打散重组：移花接木公式

## 金鱼公式

**爆款 = 爆款脚本 + 爆款开头 + 分镜画面信息密度**

## 移花接木实战

| 来源 | 素材 | 播放量 |
|------|------|--------|
| Zero大佬视频 | 跳飞机开头（视觉钩子） | 1.7亿 |
| 山海经爆款 | 机器人牺牲自己救卡布奇诺（脚本） | - |
| AG缝合版 | 跳飞机开头 + 山海经脚本 | 1000万 |

## 关键

- 嫁接点逻辑自洽：跳飞机去世 → 完美衔接悲伤故事
- 好脚本换开头就能焕发新生，不要用一次就扔
- 止步千万是因为分镜信息密度不够，不是思路问题
- 同行照抄也跑出几百万，验证思路普适
