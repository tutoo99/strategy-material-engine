# 爆款视频三要素拆解法

故事类爆款shorts = 爆款脚本 + 爆款开头 + 爆款元素堆叠

## 1. 爆款脚本分析

选择题类+形象冲突类最常见，有明显的对比：好坏、贫富、胖瘦、老幼、强弱，一个天一个地。

**选择题脚本通用结构模板**（7段式）：

1. 形象反差（如Mira让Rumi变成穷胖丑玩偶）
2. 富豪选A（如Abby选Mira）
3. A带来坏事（如Mira打翻碗撕烂衣服）
4. 穷人选B（如Jinu选Rumi）
5. B带来好事（如Rumi给鸡腿给衣服互相温暖）
6. 穷人和B通过某个途径获得道具
7. B形象升级（完成闭环）

其他变体：天堂地狱门选择题、悬崖救谁选择题、求婚选择题等。

## 2. 爆款开头分析

**开头前3秒抓眼球三法**：
1. 爆炸性画面或音效：砸玻璃、尖叫、红色警告、爆炸
2. 镜头推进运镜：更靠近画面让观众贴近信息
3. 快速转场：2s出两画面，第一个可平淡但第二个要足够爆炸反转

**核心公式**：警告 + 形象反差（美丽变丑陋/富有变贫穷/瘦子变胖子）

## 3. 爆款元素分析

**判断方法**：多看爆款视频，里面重复出现的元素就是爆款。

常见爆款元素：美人鱼、砸玻璃、吃钻石、假发、光头、超长头发、药水、躲垃圾桶、穷人打工赚钱等。

**形式创新也是爆款元素**：针织玩偶形象最早出自切片视频，被徐徐改编成KPOP版后大爆。
