# 运镜本质与推拉深层逻辑

## 五种基本运镜

| 运镜 | 动作 | 核心用途 |
|------|------|---------|
| 推 | 镜头靠近主体 | 聚焦、进入角色内心、放大情绪 |
| 拉 | 镜头远离主体 | 交代环境、揭示信息、格局转换 |
| 摇 | 镜头原地旋转 | 交代横向空间关系、时间流逝 |
| 移 | 镜头平移 | 跟随人物、平行叙事 |
| 跟 | 镜头跟踪人物 | 增加真实感、呼吸感 |

## 推拉深层逻辑（原子核心观点）

原子强调：推和拉的区别**不在于情绪方向**（不是"推=紧张、拉=放松"这种简单对应），而在于**"服务于谁"**。

**推镜头的本质**：框越来越小，环境被排除，观众被迫只看主体。推镜头服务于**被推到的那个角色**——观众的关注点被强制引导到这个人身上。

**拉镜头的本质**：框越来越大，新信息涌入画面。拉镜头服务于**被拉出来的新角色/新空间**——观众看到的不只是原来的主体，还有更广阔的格局。

### 推镜头三步走
1. 起幅：交代环境，让观众知道在哪
2. 推进：逐步排除干扰，聚焦到主体
3. 落幅：框紧主体，放大情绪

### 实例分析

拉片实例：女生撞到男生场景中，女生的镜头在推（表达她情绪变化），但男生的镜头**不推**，给固定中景。原因：接下来要接女生脸红的大特反应，如果男生也推了，两个推镜头会让观众分不清到底该关注谁。男生的固定镜头是给女生的特写"让路"。

## 不该推的时候别推

原子在拉片《我的治愈系游戏》中特别指出：女人吃生肉的镜头完全没有推拉摇移。原因：
- 这里传达的是**事件**（吃生肉）而非情绪
- 画面本身已有视觉冲击力，推镜头反而会剥夺观众对画面的第一眼感官
- 平铺直叙的固定镜头让残酷感直给，反而更震撼

对比：男主反应镜头立刻推了，因为这里要激发的是男主的恐惧情绪。

## 跟镜头的呼吸感

跟镜头让摄影机"活"起来，观众感觉自己也在场，增加了叙事的真实感和临场感。动态漫中跟镜头可以用来模拟人在空间中的移动感。
