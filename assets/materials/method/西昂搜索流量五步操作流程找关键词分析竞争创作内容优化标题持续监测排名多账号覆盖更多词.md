## 五步流程

```
找关键词 → 分析竞争 → 创作内容 → 优化标题 → 持续监测
   ↑                                         ↓
   └────────── 挖掘新关键词循环 ───────────────┘
```

## 进阶策略

1. 不断挖掘新关键词提高账号权重
2. 多账号覆盖更多领域（像多地开多店）
3. 内容质量 > AI生成（平台有识别限制）

## vs 网站 SEO

底层逻辑相同：选关键词 → 做内容 → 拿排名 → 持续流量
