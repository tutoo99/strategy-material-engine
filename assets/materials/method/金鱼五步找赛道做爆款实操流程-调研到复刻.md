# 金鱼五步找赛道做爆款

## 1. 找赛道

- 新手最大坑：自己臆想不做调研
- 标准：能拿到结果的 + 能赚钱的 + 能用AI模仿的
- 工具：YouTube Monkey插件看频道流量稳定性、预估RPM、倒推盈利
- 符合收入预期就切入

## 2. 找爆款对标

- 搜索赛道关键词（如Kpop赛道搜Kpop/rumi/golden）
- 找到关键词下有多个爆款且无违规的频道作为对标
- 每天看对标新发视频的流速（每小时增长观看次数）
- 流速快马上跟，第一时间吃第一波流量

## 3. 逐帧拆解对标

- **AI无法精准分析细节，必须自己拆**
- 放到剪映逐帧分析：分镜、画面、色调、饱和度、场景、角色位置、动作、运镜方式、音效、剪辑节奏
- 一条59秒的视频花数小时拆解，拉滑动帧一帧一帧看

## 4. 一比一复刻

- 先像素级拆解分析，一比一复刻
- 目的：熟悉平台规则 + 掌握AI视频制作流程 + 理解爆款逻辑
- 熟练后再思考优化超越 = **先抄后超**

## 5. 搭建爆款库

- 每日拆解后记录到爆款库
- 分类：脚本库 / 开头库 / 元素库等
- 日拱一卒，量变形成质变
