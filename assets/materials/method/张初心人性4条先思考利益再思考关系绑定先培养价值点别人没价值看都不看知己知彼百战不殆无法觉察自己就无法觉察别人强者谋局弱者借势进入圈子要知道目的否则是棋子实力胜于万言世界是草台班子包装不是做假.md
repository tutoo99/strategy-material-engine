## 人性4条

```
1. 先利益后关系
   世界=价值交换不是靠关系好
   连父母都不无条件爱你
   → 先培养价值点再聊关系

2. 知己知彼
   不知道自己要什么=不知道别人要什么
   人际不好=无法觉察自己=无法觉察别人

3. 强者谋局弱者借势
   进圈前=先给自己谋一个局
   不知道想从谁那得到什么=棋子
   德州：不知道赢谁的钱=输钱的人

4. 实力胜于万言
   世界=草台班子
   包装≠做假=列优势避免劣势
   身份都是自己给的
```
