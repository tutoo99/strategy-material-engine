## 号获取方式

| 方式 | 数量 | 分成 | 备注 |
|------|------|------|------|
| 朋友号（个人） | ~20 | 9:1 | 早期阶段 |
| 大学校长 | 200+ | - | 学生实习账号 |
| 航海家毛老师 | 200+ | - | 合伙后资源 |
| **合计** | **500+** | | |

## 9:1分成模式

- 朋友只需要注册号+提供账号
- 刘智行负责运营、内容、变现
- 收入9:1分成（朋友拿1成被动收入）
- 朋友月入几百到上千

## 踩坑：家人号

拿妈妈号被骂搞诈骗→妈妈打电话给老婆让她防着→不再用家人号
教训：有代沟的人不适合合作，跟同圈层朋友解释清楚更高效

## 可借鉴

三哥要拿号也可以走这条路：找到信任的朋友提供号，给被动收入分成。同时可以通过生财圈链接到有号没方法论的人。
