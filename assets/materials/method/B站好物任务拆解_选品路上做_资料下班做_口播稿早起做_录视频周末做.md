# 任务拆解与时间规划

## 核心原则

把任务拆成独立小任务，根据实际情况合理安排时间。

## 任务拆分

| 任务 | 建议时间段 | 适合场景 |
|------|-----------|---------|
| 选品（品类+商品） | 上下班路上 | 通勤时间 |
| 搜集整理资料 | 下班回家或周末 | 碎片时间 |
| 写口播稿 | 早到单位一小时 | 提前到公司 |
| 录制视频（开头结尾） | 周末 | 需要安静环境 |
| 录制音频（中间部分） | 下班回家 | 晚上时间 |
| 剪辑 | 下班回家或周末 | 大块时间 |

## 辰木的时间分配

- 通勤20分钟：刷对标账号/搜集资料
- 午休30分钟：改稿子
- 接孩子路上20分钟：手机端操作
- 晚上21:00-22:30（90分钟）：大块副业时间
