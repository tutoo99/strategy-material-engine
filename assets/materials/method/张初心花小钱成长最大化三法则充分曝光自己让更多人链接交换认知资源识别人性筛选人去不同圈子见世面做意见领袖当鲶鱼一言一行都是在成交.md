## 三法则

```
1. 充分曝光
   → 每次发言加几百人
   → 交换认知/资源
   → 识别人性筛选合作对象

2. 见世面
   → 组局吃饭了解项目（不是为成交）
   → 无法通过想象创造商业模式
   → 见世面=好品味=好东西

3. 成为意见领袖（鲶鱼效应）
   → 和冠军组在一起或成为冠军
   → 把自己当鲶鱼放进组里转起来
   → 一言一行都在成交
```
