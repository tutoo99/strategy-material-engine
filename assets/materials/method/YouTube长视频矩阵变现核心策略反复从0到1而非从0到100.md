## 反常识核心：反复0到1 不是 0到100

大多数人以为找到一个好赛道后复制10个号=10倍收益。怅惘实测结论：同一赛道多号会互相抢量，全都活不长。

## 正确的矩阵策略

1. **多号起步**：先做5-6个号用数量怼概率
2. **多选题测试**：不同号测不同选题，东边不亮西边亮
3. **每赛道限1-2号**：找到好赛道只留1-2个号，然后找下一个赛道
4. **大量筛优**：从30-60个号里筛出10个千刀号

## 月入万刀的数学

```
10个号 × 平均1000刀/月 = 10000刀/月
10个号从30-60个号里筛出
号与号之间用波动互补实现整体稳定
```

## 时间效率

- 怅惘每天只工作2小时，月入5700刀（2026年3月）
- 万刀目标不需要增加工作时长，只是方法论复制2-4遍
- 眼光正确时新号前3-4条就应见千次播放

## 眼光的判断标准

正常新号如果眼光正确，前3-4条就应该见到千次播放。如果苦哈哈做了十几条还几百播放，问题大概率在赛道选择而不是制作水平。
