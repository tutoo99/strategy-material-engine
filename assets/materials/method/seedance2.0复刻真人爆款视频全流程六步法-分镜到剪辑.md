# seedance2.0复刻真人爆款视频全流程六步法

## 第一步：分镜设计(Gemini 3.1 Pro Preview)
- 喂视频链接+提示词给Gemini
- AI输出事件总结→确认正确→继续→生成分镜提示词
- 必须对照原视频检查：AI可能加分镜也可能减分镜

## 第二步：分镜修正
- 统一主角称呼(如Mommy改成Nezuko，因为人物关系AI会搞错)
- 运镜检查：情绪镜头拿不准就用推不用拉
- 删掉AI自作主张加的分镜(案例中27-35镜都是AI加戏)
- 补充AI遗漏的分镜(直接和AI说让他补充提示词)

## 第三步：角色参考图
- Gemini Create Image上传参考图生成角色图
- 版权规避：换服装(让豆包设计乞丐穿搭)+提示词不能出现动漫角色原名
- 人脸问题：转彩色素描风格可避人脸检测

## 第四步：生成视频(seedance2.0)
- 截图分镜表格，10个分镜为一段
- 上传角色参考图保证角色一致性
- 无台词分镜用seedance 1.0(不会卡真人)
- 提示词含景别运镜场景人物动作神态+配音台词

## 第五步：补分镜
- 检查生成结果是否有跳切遗漏
- Gemini上传关键帧+角色参考图生成补充分镜图
- 用seedance 1.0生成补充分镜视频

## 第六步：剪辑合成
- BGM：YouTube自动识别/QQ音乐听歌识曲/问Gemini
- 分镜时长：半秒到一秒半，信息镜头短情绪镜头长
- 删冗余镜头：不影响剧情理解+删后衔接更丝滑的都删
- 补充音效：剪映音效库搜索匹配
