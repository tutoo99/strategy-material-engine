## 找基因三步法

```
1. 做大量人格测试 → 发现倾向
2. 问身边人对你的反馈 → 外部视角
3. 回顾最顺手的时刻 → 内部验证

判断标准：
那件事对你太简单了
→ 简单到你以为人人都会
→ 那就是你的基因
```
