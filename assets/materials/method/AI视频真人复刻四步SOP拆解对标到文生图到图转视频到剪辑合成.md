# AI视频真人复刻四步SOP

## 第一步：拆解对标视频

1. 复制对标视频链接粘贴到 Google AI Studio
2. 让AI输出每个分镜的文生图提示词和图转视频提示词
3. 检查分镜有不对的继续修改
4. 确定角色形象、场景、风格时**第一步永远是讨论**，讨论完再输出新提示词

## 第二步：文生图（即梦）

1. 打开即梦，选择模型和比例
2. 用即梦批量上传插件添加分镜
3. 上传参考图(最多4张)和提示词
4. 点击开始上传，等待收菜

## 第三步：图转视频（Vidu）

1. 打开Vidu，选择图生视频，模型选Q2
2. 用Vidu批量上传插件批量上传分镜图
3. 一行提示词对应一张图片
4. 不满意的单独修改重新抽卡

## 第四步：剪辑合成（剪映）

1. 按顺序放到剪辑轨道
2. 背景音乐参考对标视频，shorts 1分钟以内音乐随便用
3. 配音用剪映自带：先识别字幕，再选择朗读
4. 导出发布

## 全套AI工具链

Gemini拆解脚本和提示词 → 即梦/豆包出图 → 可灵/海螺/即梦/Vidu图转视频 → 剪映剪辑
