## AI时代财富公式

```
细分赛道 + 垂直人群 + 标杆产品 + 极深护城河
→ 锁定一群人 → 持续深耕 → 持续优化 → 占领心智
→ 结构形成 → 极难推翻（AI帮你筛得越来越精准）
```

## 反模式（必死）

```
大赛道 + 大众产品 + 宽泛人群
= 方向与时代相反
= 逆着洪水跑，跑得越快死得越快
```
