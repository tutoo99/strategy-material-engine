## 识破四类人

```
1. 赚钱比你多的人
   → 研究入场时间+产品+核心技能
   → 找最赚钱的人研究+交朋友

2. 离你最近的人
   → 看同事/同学/父母/亲戚
   → 经济水平=周围6人平均值
   → 有些时候赚不到钱=身上太多贫穷基因

3. 喜欢倒腾项目的人
   → 赚钱风向标
   → 分析：做什么/不做什么/为什么不做
   → 通过别人项目吸取经验少走弯路

4. 比你先走几步的同龄人
   → 早熟+早闯荡+靠自己打拼
   → 搞清楚赛道/进场/成绩/努力/时间
```

## 核心

```
赚钱不是要多努力
而是知道哪些地方用力事半功倍
```
