## 两种模式

```
卖产品（低宱）          卖身份（高宱）
解释价值                 成为价值
告诉别人你有多好        让他感觉跟你在一起他在变好
讲功能/交付/细节        卖判断/理解/路径
卖成本逻辑               卖意义逻辑
靠证明                   靠认同
纠结的人问细节            看了很多轮直接成交
```

## 核心公式

```
用户买的不是：你为他做了多少
用户买的是：他成为了什么

低宱靠证明（我能做什么）
高宱靠认同（我们是一类人 + 我能带你到什么状态）
```
