## 制作流程

```
确定风格 → 人物设计(三视图16:9) → 场景设计(21:9)
→ AI生成分镜视频(vidu多参) → 配音 → 对口型 → 剪辑
```

## 工具链

| 环节 | 工具 |
|------|------|
| 生图 | 即梦/banana/MJ |
| 视频生成 | vidu多主体参考 |
| 配音 | index-TTS本地 / minimax / 可灵 |
| 对口型 | LatentSync本地 / 可灵2.6云端 |
| 剪辑 | 手动（剪映等） |

## 剪辑避坑

- 景别跨度不能太大（远→中→近→特）
- 要有呼吸感
- 不留多余画面
- 画面叙事要完整
