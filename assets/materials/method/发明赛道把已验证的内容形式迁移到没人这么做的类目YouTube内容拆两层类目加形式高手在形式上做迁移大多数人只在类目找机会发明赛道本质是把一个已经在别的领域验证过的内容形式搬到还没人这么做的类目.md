## 发明赛道

```
YouTube内容 = 类目 + 形式

❌ 多数人：只在类目里找机会（健康/历史/健身）
✅ 高手：在形式上做迁移

迁移公式：
  已验证的爆款形式 → 没人这么做的类目 = 新赛道

实例：
  健康：专家坐讲 → 高质量动画人体机制图解
  历史：平铺直叙 → 纪录片旁白+悬疑钩子+电影转场+地图
  爱泼斯坦：八卦阴谋论 → 硬核金融犯罪拆解（Capital Case）

本质：
  内容还是那个内容 → 呈现方式变了 → 位置就变了
  YouTube = 做位置的竞争，不是做内容的竞争
```
