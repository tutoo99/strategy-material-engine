# B站好物选品四步法

## 方法一：大海捞针（看销量+佣金+竞争度）

1. 打开京东首页，按品类逐个看
2. 看京东产品销量：热销几百万、几十万，普通几万也行
3. 看京粉佣金：一单能赚20块以上，小众品类越高越好
4. 在B站搜该产品，看第一页近几天发的视频：
   - 播放量几百几千的多 → 竞争不激烈（B站会给新视频机会）
   - 80%都是几万几十万播放 → 不好挤进去

## 方法二：找细分领域

大品类挤不进去时，缩小范围：
- 电冰箱 → 三门电冰箱 / 法式电冰箱 / 美的电冰箱 / 海尔电冰箱
- 搜索下拉菜单里有的词 = 有流量，先抓精准流量

## 方法三：跟着热点跑

不是新闻热点，是搜索热点：
- 季节性：夏天降温、冬天取暖
- 节日：过年送父母手机推荐
- 重要日子：开学季针对大一新生选品
- 实在想不出可以问AI

## 方法四：对标写啥我写啥

- 建在线文档，收集优质对标账号
- 优先找AI味重、什么品类都发的作者（播放量数据真实）
- 如果某个视频流量异常好（平时几百，突然两万多），重点研究这个品类
