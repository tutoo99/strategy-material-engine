# B站好物找商品方法

## 分类方式（三选一）

1. **按价格分类**（最常用）：2000以内4款 + 2000-3000元4款 + 3000以上4款
2. **按特征分类**：对开门4款 + 十字门4款 + 法式4款
3. **按品牌分类**（仅适用于品牌少的品类，如电话手表）：小天才4款 + 华为4款

## 选品四原则

1. **覆盖常见品牌**：每个分类尽量覆盖主流品牌。观众点了你推荐的海尔没买但买了同店其他款，你也有佣金
2. **销量要高**：选京东自营店或品牌旗舰店，不推荐第三方店铺（下单率低，观众也不信任）
3. **佣金要合适**：销量和口碑差不多时，优先选佣金高的
4. **尽量有差异**：同价位选有特色的款（空间大、自研保鲜技术等），推荐时有侧重点

## 最常用工具

京东【排行榜】，从头到尾看每款的佣金。更简单的办法：直接看对标账号挂什么商品。
