# AI短剧单集制作成本数据

| 项目 | 工具 | 单价 | 说明 |
|------|------|------|------|
| 生图 | Nano Banana Pro / 即梦V4 | 0.045元/张 | 角色参考图+场景参考图+分镜图 |
| 视频生成 | SORA2 | 0.06元/个 | 图生视频，支持音画同步 |
| 视频备选 | VEO3 | 未公开 | SORA2不理想时切换 |
| 配音 | SORA2/VEO3自带 | 包含 | 音画同步生成语音+音效 |
| 配音备选 | MiniMaxi / ElevenLabs | 按需 | 更高质量声音控制 |
| 后期 | 剪映 | 免费 | 粗剪+音频分离+字幕 |

## 单集成本估算
- 13个分镜 + 角色参考图(5张) + 场景参考图(4张) + 抽卡重试
- 生图总费用约1-1.5元，视频总费用约1元
- **单集总成本不超过2元**

## 时间估算
- 零基础初学者完成单集约需2小时
- 雨文此前已有YouTube AI视频矩阵经验（1个月4个YPP），上手更快
