# AG 半年数据复盘

## 时间线

| 时间 | 事件 |
|------|------|
| 2024.11 | 加入生财有术 |
| 2025.6 | 航海爆2000万播放，通关高级YPP |
| 7-9月 | 死卡万播两个月 |
| 9.15 | 富太太IP出千万爆款 |
| 10月 | 2个频道YPP被取消（儿童危险），损失3500刀 |
| 11月 | 新频道继续千万爆款 |
| 2月 | 新频道遭同质化限流 |

## 核心数据

- 总播放：5亿+
- 高级YPP：3个
- 总收益：约1万刀
- 单条最高播放：8000万（但YPP已被取消，零收益）
- 富太太IP不到1个月跑出1亿播放
