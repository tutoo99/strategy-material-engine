# 抖音心理测试小程序分佣数学模型

## 单视频收益计算

| 指标 | 数值 | 说明 |
|------|------|------|
| 点赞数 | 10w | 爆款鸡汤视频 |
| 赞播比 | 1:30 | 播放量约300w |
| 播放到打开率 | 5% | 约15w打开小程序 |
| 打开到付费率 | 7% | 约10500人付费 |
| 客单价 | 1元 | 小额付费心理测试 |
| 佣金比例 | 80% | 分佣者拿80% |
| 单爆款收益 | 8400元 | 分佣者净收入 |

- 日产100个低成本鸡汤视频
- 假设爆2个
- 日收益：16800元（理想情况）
