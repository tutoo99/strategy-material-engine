# 天才老师双Agent矩阵核心数据

| 指标 | 数据 |
|------|------|

| 总订阅 | 1.3M |
| 总播放 | 2.25亿 |
| 总视频 | 92条 |
| 运营时长 | 3个月 |

- 41条视频，10W+播放25条，百万14条，千万1条

- VPS（煤老板）：~36元/月
- API调用（拼车）：300元/月
- 即梦会员：299元/月
- Mac Mini折旧：~130元/月
- 合计：~765元/月
