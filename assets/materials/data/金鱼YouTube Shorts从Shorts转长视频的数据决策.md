# 金鱼Shorts转长视频的数据决策

## 关键数据

| 指标 | 数值 |
|------|------|
| 3个月总观看 | 过亿 |
| 3个半月累计变现 | 2000+美金 |
| Shorts天花板年入 | ~1400万人民币(MrBeast级) |
| 长视频目标3年年入 | 500万元 |

## Shorts天花板算式

MrBeast级年播放1000亿 → 互动播放×0.5=500亿 → 泛娱乐RPM 0.04 → 500×10000×10×0.04=200万美金/年≈1400万人民币

## 制作成本

| 项目 | 成本 |
|------|------|
| flow账号(闲鱼) | 45-50元/月，45000积分 |
| Veo3.1 fast | 10积分/条 |
| Veo3.1 quality | 100积分/条 |
| 10分钟长视频(300分镜) | ~3.6元(fast) |
| 含抽卡+网络配置 | 几十块不到 |

## 推流规律

- 爆款视频可能20多天才开始推流
- 平台每隔一段时间断流，内容达标后会把流量推回来
- 每天按时出摊是关键
