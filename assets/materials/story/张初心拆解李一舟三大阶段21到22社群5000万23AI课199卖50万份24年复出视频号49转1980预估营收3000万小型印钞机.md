## 李一舟三阶段

```
第一阶段（21-22）社群模式      → 5000万
第二阶段（23）    AI课199元    → 50万份
第三阶段（24下半年）视频号      → 5万咨询 → 49转1980 → ~3000万
```

## 单元模型=小型印钞机

```
49元（前端） → 5%-15%转化 → 1980元（后端）
至少10万单 × 1980 = ~2亿流水
一年半预估营收 ~3000万（扣除转化率偏低）

纯线上 → 容错大 → 能跑动ROI就做
先极高客单试水 → 转化不高 → 降客单做普通人生意
```
