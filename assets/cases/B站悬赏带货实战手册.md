# B站矩阵玩家鱼丸-悬赏带货蓝链实战-60人团队跑通多品类变现

---

## 【案例身份证】

**帖子标题：** B站好物：实战手册（官方指导+案例）
**原文链接：** 生财有术内部帖（https://scys.com/articleDetail/xq_topic/2852485555428851）
**来源平台：** 生财有术
**是否商业化验证：** 是（作者自称取得过结果，团队60人在跑）

**一句话业务：** 一个2022年入局B站的矩阵玩家，通过B站官方悬赏带货（评论区蓝链）跑通图书、零食、服饰等多品类CPS变现，8个账号2个主号各70w粉，带60人团队实操，并整理出从账号准入到选品到内容到运营的完整SOP。

**作者是谁：**
- 网名鱼丸，生财有术圈友
- 2022年开始做B站，目前拥有8个账号
- 2个主力账号各70w粉
- 团队60人在跑B站带货项目
- 实操品类：图书、零食、服饰
- 2023年做过B站直播带货（迷瞪时期），测过无人直播，已验证现在不行

**启动资源：**
1) **花了多少钱？** 原文未明确金额，但提到"这里没有百万GMV"，暗示投入不大，门槛可控
2) **投入了多少时间？** 2022年入局持续至今（约3年+），前期全职投入
3) **有没有合伙人？** 有，团队60人在跑，属于团队矩阵玩法
4) **用到了什么特殊技能或设备？** 视频制作（可用AI提效）、SEO优化能力、电商选品能力、淘客联盟操作经验
5) **其他启动条件：** 粉丝量≥1000即可开通悬赏权限，30天内发过原创视频，年满18周岁

**核心目标：**
- 拆解B站悬赏带货（蓝链带货）的完整操作流程

## 【决策地图】

**1. 决策点：为什么选B站带货而不是抖音/小红书？**
- **选择：** 入局B站悬赏带货
- **依据：** B站开放电商跳转口允许评论区挂蓝链，平台政策友好；视频SEO友好能获得长期精准流量形成流量复利；AI提效降低了视频制作门槛；平台公开TOP100带货博主榜单数据透明可复盘；类似知乎好物的窗口期机会；中腰部达人靠带货头部靠商单，中腰部有生存空间
- **动作：**
  1. 注册B站账号，确保粉丝≥1000
  2. 30天内发布至少1条原创视频
  3. 年满18周岁后，在B站APP→我的→推荐服务→商业服务→悬赏计划，点击申请加入
  4. 或PC端进入 member.bilibili.com/v2#/allowance/mutual_up 点击【申请加入】
- **工具：**
  - B站APP
  - B站创作中心
- **参数：**
  - 粉丝量≥1000
  - 30天内有原创视频
  - 年满18周岁
- **是否推测：** 否
- **证据摘录：** 平台政策好：B站开放电商跳转口，允许创作者把流量带走卖货，评论区挂蓝链等，为带货提供便利。流量有优势：视频SEO友好，能获得长期精准流量。制作门槛低：有AI提效。带货数据透明：平台公开TOP100带货博主榜单。认为这是一个类似知乎好物的好机会。

**2. 决策点：选什么品类带货？**
- **选择：** 图书 + 零食为主（个人实操），同时分析了3C、服饰、家装、美妆等品类
- **依据：** 图书：佣金相对较高、拍摄成本低（书单号）、退货率低。零食：不需要出镜、退货率低。3C数码：案例最多、市场大、B站科技测评生态好但竞争激烈。服饰：退货率高，直接直播带货已不行（2023年可现在不行），需走悬赏带货。家装：高客单价利润高但需专业度。美妆：带货普及度高但需颜值和真实使用体验。蓝海类目：运动户外、二奢、图书音像
- **动作：**
  1. 打开B站APP→我的→商业服务→悬赏带货→选品中心
  2. 优先看"分区热卖"和"全站热卖"榜单，关注销量指标
  3. 新手建议从图书或零食入手：佣金高+退货低+制作门槛低
  4. 匹配"粉丝喜好+自身定位"，粉丝喜好匹配度优先级高于佣金
  5. 关注客单价：100-300元更匹配B站用户消费能力，ROI更易达成
- **工具：**
  - B站选品中心
  - 火烧云数据（查看TOP100带货博主榜单）
- **参数：**
  - 100-300元客单价
  - 佣金率不低于20%
  - 退货率低的品类
- **是否推测：** 否
- **证据摘录：** 看过我上一个帖子，就知道我是做图书类目、零食类目的，这俩其实，我个人感觉是最好做，零食不需要出镜，退货相对较低，图书是佣金相对较高。蓝海类目：运动户外、二奢、图书音像。

**3. 决策点：怎么挂蓝链开始赚钱？**
- **选择：** 通过B站悬赏计划绑定淘宝联盟/京东联盟PID，在评论区挂蓝链
- **依据：** 评论区蓝链是转化效果最好的资源位，更符合社区属性；B站平台不抽佣，联盟服务费按联盟规则收取（淘宝联盟收佣金10%）；可在视频投稿时配置，也可在已发布视频的评论区配置；每条评论可添加20件商品，评论数量无限制
- **动作：**
  1. 开通悬赏权限后，立即绑定淘宝联盟PID：打开 pub.alimama.com 注册淘宝联盟账号→创建推广位→获取PID
  2. 回到B站创作中心→收益管理→悬赏计划，绑定PID（每日限改1次）
  3. 商品入库：在选品中心通过搜索/榜单/推荐/链接识别四种方式添加商品到选品车
  4. PC端挂蓝链路径：创作中心→收益管理→悬赏带货→选品车管理→选择商品→去带货→选择稿件→评论区蓝链→编辑文案→发布
  5. 在视频播放页评论区找到该条评论，点击"置顶"
  6. 视频投稿时务必勾选【商业推广】→【视频带货】，否则被判逃单会限流
- **工具：**
  - 淘宝联盟 pub.alimama.com
  - 京东联盟
  - B站创作中心-悬赏计划
- **参数：**
  - 每条评论最多20件商品
  - PID每日限改1次
  - 淘宝联盟收佣金10%技术服务费
- **是否推测：** 否
- **证据摘录：** B站蓝链带货其实就是一个B站通往站外的跳转链接。通常出现在评论区，在置顶评论里的蓝色链接。在蓝链的前面会有一个小标，一般是"淘"/"JD"的字样。TIPS：目前评论区挂接悬赏链接是相对转化效果不错的资源位。

**4. 决策点：怎么做出能带货的视频内容？**
- **选择：** 采用"人设稳定 + 创意模型 + 选题结构化"的内容生产方式
- **依据：** B站用户以Z世代和都市白领为主，主张悦己和理性消费；不需要高制作成本，有AI提效；三种内容方向验证有效：痛点安利、知识科普、使用教程；人设必须与日常内容统一不能频繁切换
- **动作：**
  1. 先确定创意模型（三选一）：代入型（好物分享/开箱，自身是目标人群）、专家型（专业领域科普）、评测型（深度测评对比）
  2. 按内容结构模板写脚本：痛点安利（抛痛点→解决方案→卖点对比→引导转化）、知识科普（科普→产品讲解→引导转化）、使用教程（产品介绍→操作教学→使用技巧）
  3. 视频封面：突出重要信息，风格及字体统一，结合节日用色彩和关键字
  4. 资源位优先选评论区置顶蓝链（社区属性好+持续展示+可互动）
  5. 蓝链标题优化：品牌词+属性词+类目+卖点+优惠信息+达人词
  6. 口播引导：视频中进行口播或字幕引导粉丝去评论区点蓝链
- **工具：**
  - AI视频制作工具（剪映等）
  - 稿定设计（封面制作）
- **参数：**
  - 视频时长建议5-15分钟（B站长视频优势）
- **是否推测：** 否
- **证据摘录：** 做电商的，懂得都懂。有AI提效，平台属性也降低了视频制作门槛，像写朋友圈一样就能做出不错的视频。内容方向：痛点安利（抛痛点→方案→卖点对比→引导转化）、知识科普、使用教程。

**5. 决策点：怎么提升带货转化数据？**
- **选择：** 按GPM公式拆解，从"人货场"三维度分别优化
- **依据：** GPM = GMV/视频vv*1000 = 商品点击率 × 下单转化率 × 客单价 / 视频vv × 1000。商品点击率及视频vv优化"人"（IP定位、内容结构、创意、蹭热点、口播话术）。客单价优化"货"（选品结构、价格区间、爆品打造、粉丝画像匹配）。下单转化率优化"场"（营销氛围、活动福利、粉丝互动运营）
- **动作：**
  1. 人-流量优化：搭配SEO关键词，利用B站搜索长尾流量
  2. 人-内容优化：封面突出信息+口播不棒读+卖点呈现清晰
  3. 货-选品优化：100-300元客单价+佣金率不低于20%+退货率低的品类
  4. 货-结构优化：爆款引流品+利润品+复购品组合
  5. 场-营销优化：跟大促节点走（618/双11/年货节）+设置粉丝专享日
  6. 场-互动优化：评论区置顶蓝链+动态发福利抽奖+粉丝群运营
- **工具：**
  - 火烧云数据（数据分析）
  - B站数据后台
- **参数：**
  - GPM = GMV/视频vv*1000
  - 客单价100-300元
  - 佣金率≥20%
- **是否推测：** 否
- **证据摘录：** 搭配SEO，信息流。官方内部群——优质信息差。类目：3C数码、家居、服饰、美妆、零食、图书（注意顺序）。其次看招聘信息，一个大公司，要看裁员那个部门，小公司看招什么人。

**6. 决策点：怎么查看收益和提现？**
- **选择：** 淘宝联盟/京东联盟收益在联盟端查看，会员购在B站端查看
- **依据：** 淘宝联盟和京东联盟的带货收益B站暂不显示，需去联盟端查看；会员购联盟用户确认收货后T+15天结算，次月5-10日打款至贝壳；淘宝联盟收佣金10%技术服务费
- **动作：**
  1. 淘宝联盟收益查看：登录 pub.alimama.com → 效果报表 → 订单明细表 → 下载报表
  2. 京东联盟收益查看：登录京东联盟后台查看
  3. 会员购收益查看：悬赏带货→我的收益→贝壳提现
- **工具：**
  - 淘宝联盟 pub.alimama.com
  - 京东联盟
  - B站悬赏带货-我的收益
- **参数：**
  - 会员购T+15天结算
  - 次月5-10日打款
  - 淘宝联盟技术服务费10%
- **是否推测：** 否
- **证据摘录：** 淘宝联盟/京东联盟的带货收益B站暂不显示。会员购联盟：用户确认收货后T+15天结算，次月5-10日打款至贝壳。淘宝联盟：收佣金10%技术服务费。

## 【核心心法】

### 作战三原则：

1. **品类选择优先看"退货率+佣金率+制作门槛"三指标，粉丝匹配度高于佣金高低。** 不追爆款品类，追你能稳定输出的品类。新手从图书或零食入手最稳。

2. **评论区蓝链是B站带货的核心资源位，视频投稿时必须勾选商业报备。** 未勾选"视频带货"会被判逃单限流。评论区蓝链+置顶+口播引导，是转化效率最高的组合。

3. **内容必须先立人设再带货，人设不稳定等于没有信任基础。** 选定代入型/专家型/评测型其中一种，持续输出。不要一条卖零食下一条卖手机壳。

### 最大一个坑：
- **坑点：** 视频投稿时忘记勾选"商业推广→视频带货"，被判逃单限流，后续视频推荐量断崖下跌
- **解决方案：** 如果暂时不确定带什么品，投稿时选"投稿后再配置商品"选项先报备；发布后5分钟内及时配置蓝链，不要拖延导致流量流失

### 最值钱一句忠告：
**"中腰部靠带货，头部靠商单。"** B站带货的窗口期不是给百万粉大UP的，而是给1000粉起步、能稳定产出内容、懂选品的中腰部玩家的。平台流量复利机制（视频长期可被搜索到）意味着早期视频可以持续带来佣金，这是B站带货区别于抖音的核心优势。
