# 【案例名称】YouTube航海玩家大臣——真人AI爆款复刻打法，1年做出8个高YPP、月入4500刀

---

## 【案例身份证】

**帖子标题：** YouTube新人攻略：真人脚本挖掘以及如何用seedance2.0复刻出一条40秒真人爆款视频  
**原文链接：** https://my.feishu.cn/docx/B2LodJh0RouFpNxT2zGc0R93nLh  
**一句话业务：** 通过“爆款IP × 爆款脚本”的方法在 YouTube 做真人 AI 视频，先验证流量，再用 Gemini + Seedance + 剪映复刻爆款短视频，目标是开 YPP 并赚美金。  
**作者是谁：** 大臣，去年 3 月参加 YouTube 航海后入局的 AI 视频创作者。

**启动资源：**
1) **花了多少钱？** 原文未披露总投入；免费工具包括 `Viewstats`、`Gemini AI Studio`、`Gemini 图片生成`、`剪映`、`QQ音乐听歌识曲`，主要付费项是 `Seedance` 生成额度。  
2) **投入了多少时间？** 从去年 3 月入局到分享时约 1 年；日常动作包括刷 B 站梗百科、刷 YouTube 无痕首页、持续跟踪全球电影和游戏资讯。  
3) **有没有合伙人？** 原文未提团队协作，按表述更像作者单人推进。  
4) **用到了什么特殊技能或设备？** 需要持续找 IP、筛脚本、读懂镜头语言、用 Gemini 拆分镜、用 Seedance 生视频、用剪映做节奏和音效。  
5) **其他启动条件：** 需要一个干净的小号去喂 YouTube 推荐，需要基础英文理解能力去搜海外视频，需要能长期刷平台建立内容直觉。

**核心目标：** 教新人掌握 YouTube 真人 AI 爆款复刻的基本功：找到有量 IP、找到可复爆脚本、做出一条能过平台和用户双重检验的真人 AI 短视频。  
**最终结果：** 入局约 1 年，做出 8 个高级 YPP 账号，单月盈利约 4500 刀，期间还跑出 1 条亿播爆款和 1 篇龙珠精华贴。

## 【决策地图】

**1. 决策点：新人入局 YouTube 时，先等 SOP 还是先练爆款复刻基本功？**
- **选择：** 不等现成 SOP，直接把真人 AI 爆款复刻当成基本功来练。
- **依据：** 作者认为 YouTube 和 AI 变化太快，平台不会长期允许“看到 SOP 就能稳定吃红利”的玩法，能穿越周期的是内容理解和创作基本功。
- **动作：**
  1. 先接受“没有标准答案”的前提，不把选赛道建立在别人现成 SOP 上。
  2. 以“开通 YPP、赚美金、做出爆款视频”为目标，反推自己需要补的能力：IP 判断、脚本理解、镜头拆解、生成和剪辑。
  3. 从真人 AI 爆款复刻切入，因为它同时训练选题、内容理解和生产能力。
- **工具：**
  - YouTube
  - Viewstats
- **参数：**
  - 去年 3 月入局
  - 1 年验证周期
  - 8 个高级 YPP
  - 单月 4500 刀
- **是否推测：** 否
- **证据摘录：** “不会存在‘赚钱的标准答案’……YouTube不会再允许这样的情况发生”“今天的直播，就围绕YouTubeAI视频最底层的基本功”。

**2. 决策点：爆款 IP 从哪里挖，而不是靠主观猜题？**
- **选择：** 三路并行找 IP：梗百科账号、YouTube 无痕首页、全球电影/游戏资讯。
- **依据：** 国内火的 IP 不一定国外火；旧热点和长视频热点也不一定适合当前 Shorts，必须用多个信号源交叉验证。
- **动作：**
  1. 去 B 站或抖音关注梗百科账号，定期补网络热梗和新形象。
  2. 打开浏览器无痕模式进入 YouTube 首页，先点 `Shorts`，等右侧短视频加载完，再切回首页持续刷新。
  3. 看到陌生但高频出现的形象时先截图，再用 Google 以图搜图或问 AI 确认 IP 名称。
  4. 持续跟全球电影、动画、游戏资讯，捕捉新上映或新出圈的角色与世界观。
- **工具：**
  - B站/抖音梗百科账号
  - YouTube 无痕模式
  - Google 以图搜图
  - AI 识图工具
- **参数：**
  - 3 条 IP 来源
  - 先点 Shorts 再回首页
- **是否推测：** 否
- **证据摘录：** “一个最近正在爆火的IP，油管一定会主动推给你的，所以你只需要刷就好了。”

**3. 决策点：找到热门 IP 后，是马上做还是先验证这个月还有没有量？**
- **选择：** 先验证，再开做；不做“印象中有量”的 IP。
- **依据：** 同一个 IP 会随时间、国家、视频形态变化而失效；作者举例说英雄联盟在大家印象里很大，但当月并没有高播放 Shorts。
- **动作：**
  1. 打开 YouTube 搜索目标 IP 关键词。
  2. 点击 `过滤`，依次选择 `热门程度` → `本月` → `短视频`。
  3. 观察近几天是否仍有千万或大几百万播放的视频。
  4. 进入任一相关视频，点击右上角三个点 → `说明`，再点蓝色标签查看同标签视频池。
  5. 如果近期没有足够播放量，直接放弃该 IP，换下一个。
- **工具：**
  - YouTube 搜索
  - YouTube 过滤器
  - 视频标签页
- **参数：**
  - 热门程度
  - 本月
  - 短视频
  - 近几天千万/大几百万播放
- **是否推测：** 否
- **证据摘录：** “找到一个热门IP不要马上就去做，先去搜搜看一下这个IP在油管上情况如何。”

**4. 决策点：围绕爆款 IP 做内容时，选哪一层内容深度最适合新人？**
- **选择：** 新人优先做第一层“套 IP 外壳”和第二层“轻度二创/结合人物特征”，不急着上第三层深度演绎。
- **依据：** 作者明确说对大部分 AI 创作者来说，做到前两层就够了，第三层门槛太高。
- **动作：**
  1. 打开同一 IP 的 3 条爆款样本视频，对比它们分别属于“外壳套用、轻度二创、深度演绎”哪一层。
  2. 记录每条样本里最容易复刻的角色形象、情绪桥段和反转节点，优先筛选前两层玩法。
  3. 新号先选择“借 IP 外壳演绎成熟脚本”或“结合人物特征做轻度二创”的样本开做。
  4. 等前两层跑出播放和留存后，再调整到第三层深度演绎。
- **工具：**
  - YouTube
  - 样本视频收藏夹
- **参数：**
  - 3 层内容深度
  - 先对比 3 条样本
  - 新人优先前 2 层
- **是否推测：** 是
- **证据摘录：** “对于我们大部分AI创作者来说，做到第一层和第二层就好了。”

**5. 决策点：爆款脚本从哪里找，才能避免所有人都在抄同一批油管账号？**
- **选择：** 用 YouTube 自动推荐做主线，同时从国内平台反向找脚本源。
- **依据：** 作者判断 YouTube 上的真人账号已经被对标得很厉害，只看油管容易同质化，所以要让平台推荐和国内内容池一起供脚本。
- **动作：**
  1. 准备一个没有关注过其他赛道的小号。
  2. 先关注作者列出的真人对标账号，如 `kimpro828`、`linhchihihihi` 等。
  3. 连续刷这些账号的视频，再不断刷新首页，让 YouTube 继续推相似账号。
  4. 在抖音、快手、小红书、B 站用 `搞笑`、`反转`、`欧亨利式结尾` 等关键词搜种子博主。
  5. 点进视频标签，继续顺藤摸瓜找更多脚本源账号。
- **工具：**
  - YouTube 关注列表
  - 抖音
  - 快手
  - 小红书
  - B站
- **参数：**
  - 干净小号
  - 关键词：搞笑 / 反转 / 欧亨利式结尾
- **是否推测：** 否
- **证据摘录：** “大家拿一个没有关注过其他赛道的账号，去关注下面这些账号……油管就会不断的给你推类似的账号。”

**6. 决策点：什么样的脚本值得拿来做 AI 真人复刻？**
- **选择：** 只做强画面、弱文案、跨文化可理解的脚本，并用 4 条红线做筛选。
- **依据：** 纯靠中文文案翻英文会损失效果；强文化背景的梗老外不一定能懂；违规和擦边内容还会影响 YPP。
- **动作：**
  1. 拿到脚本后先判断它是不是靠画面和情绪推进，而不是靠语言包袱。
  2. 逐条过四条红线：不做未成年人危害/虐待、不做恶心或擦边、不做强文化门槛脚本、优先强画面弱文案。
  3. 遇到“国内人秒懂、老外不一定懂”的段子，直接淘汰或重写。
  4. 只把经过筛选的脚本送进后面的分镜和生成流程。
- **工具：**
  - YouTube
  - 国内短视频平台
  - 人工脚本筛选表
- **参数：**
  - 4 条排除条件
  - 强画面弱文案
  - 跨文化可理解
- **是否推测：** 否
- **证据摘录：** “一个好的爆款脚本是爆过了还会再爆的，因为它是靠内容吸引人。”

**7. 决策点：分镜脚本用哪个工具拆，怎么避免 AI 拆错剧情？**
- **选择：** 用 `Gemini AI Studio` 拆分镜，但必须人工逐镜校对，不直接信 AI 草稿。
- **依据：** Gemini 能免费输出结构化分镜，但作者明确展示了 AI 会乱加镜头、搞错人物和运镜。
- **动作：**
  1. 打开 `https://aistudio.google.com/welcome`，确认模型选择 `Gemini 3.1 Pro Preview`。
  2. 复制对标 YouTube 视频链接，连同作者提供的三幕式提示词模板一起贴给 Gemini。
  3. 先让它输出“故事内核架构”，确认剧情主线没有跑偏后再回复“继续”。
  4. 等它生成完整分镜表后，逐镜对标原视频，删除 AI 擅自增加的镜头，补回遗漏镜头。
  5. 统一角色命名，去掉不必要的具体穿搭描述，修正台词、镜头方向和情绪节奏。
- **工具：**
  - Gemini AI Studio
  - YouTube 视频链接
  - 三幕式分镜提示词模板
- **参数：**
  - 模型：Gemini 3.1 Pro Preview
  - 分两步：先故事内核，再完整分镜
  - 发现 AI 多加 27-35 镜时删除
- **是否推测：** 否
- **证据摘录：** “检查AI是否加减分镜（如原文AI多加了27-35镜需删除）”“拿表格逐镜对标原视频剧情，修正错误”。

**8. 决策点：角色参考图怎么做，遇到版权和人脸限制怎么绕？**
- **选择：** 先用 Gemini 生成三视图；遇版权问题先换服装；遇人脸限制改成彩色素描风。
- **依据：** 直接沿用高辨识度动漫服装容易卡版权，真人脸也可能触发平台限制，需要先把可生成性问题处理掉。
- **动作：**
  1. 先在网上找到角色全身照。
  2. 打开 `https://gemini.google.com`，选择 `Create image`，一次上传一张参考图。
  3. 输入三视图提示词，要求全身、纯白背景、真人实拍风格、服装一致。
  4. 如果卡版权，就去豆包重新设计服装，例如“落魄流浪风”“随性拾荒风”。
  5. 如果卡人脸，再把图片转成彩色素描图，保留人物辨识度后继续生成。
- **工具：**
  - Gemini 图片生成
  - 豆包
- **参数：**
  - 一次一张参考图
  - 纯白背景
  - 真人实拍风格
  - 彩色素描图
- **是否推测：** 否
- **证据摘录：** “如遇版权：用豆包设计新服装”“如遇人脸问题：发送提示词‘把图片转成彩色素描图’”。

**9. 决策点：视频生成阶段，Seedance 2.0 和 1.0 怎么分工？**
- **选择：** 主流程用 `Seedance 2.0` 生成带口型台词的分段视频，缺镜或无台词镜头用 `Seedance 1.0` 补。
- **依据：** Seedance 2.0 更适合真人台词同步，但有 `10 镜 / 15 秒` 限制；Seedance 1.0 对无台词补镜更稳。
- **动作：**
  1. 把改好的分镜表按每 `10` 个分镜截成一张图。
  2. 打开 Seedance 2.0，上传角色参考图和对应分镜截图。
  3. 输入统一提示词，要求写实镜头、保留丰富音效、不要背景音乐、不要字幕。
  4. 检查生成结果是否缺镜；如果缺镜，回到 Gemini 用关键帧和角色图补生成缺失画面。
  5. 对无台词或容易卡真人的镜头，改用 Seedance 1.0 单独补生成。
- **工具：**
  - Seedance 2.0
  - Seedance 1.0
  - Gemini
- **参数：**
  - 10 镜一段
  - 15 秒限制
  - 不要背景音乐
  - 不要字幕
  - 写实镜头
- **是否推测：** 否
- **证据摘录：** “Seedance 2.0（主用，支持台词口型同步，10镜/15秒分段）+ Seedance 1.0（补无台词镜头，不卡真人）”。

**10. 决策点：后期剪辑时，BGM 和节奏怎么处理才能接近原爆款？**
- **选择：** 优先复用原视频 BGM，再按自己画面的节奏重剪，不追求 1:1 秒级复刻。
- **依据：** 作者认为 BGM 是情绪驱动核心；AI 生成镜头不可能完全对齐原片，因此要按生成结果重新找节奏。
- **动作：**
  1. 先在 YouTube 原视频下方尝试用平台自带识别找到 BGM。
  2. 若识别不到，就用 QQ 音乐“听歌识曲”或直接问 Gemini。
  3. 在剪映先铺 BGM，再把信息类镜头裁到约 `0.5 秒`，情绪镜头保留 `1-1.5 秒`。
  4. 逐镜预览，把删掉后不影响理解、还能让衔接更顺的镜头直接删掉。
  5. 打开剪映音效库，按具体场景搜索音效词，例如 `车撞`，补齐 AI 没生成的关键声音。
- **工具：**
  - YouTube
  - QQ音乐听歌识曲
  - 剪映
  - Gemini
  - youtube.iiilab.com
- **参数：**
  - 信息镜头 0.5 秒
  - 情绪镜头 1-1.5 秒
  - 先铺 BGM 再剪画面
- **是否推测：** 否
- **证据摘录：** “有BGM和没BGM剪起来完全两个感觉”“信息镜头0.5秒，情绪镜头最长1-1.5秒，根据画面节奏剪而非1:1对标原视频”。

## 【核心心法】

### 作战三原则：
1. 做任何 IP 之前，必须先走一遍 `搜索 → 过滤 → 热门程度 → 本月 → 短视频` 的验证链路，只做当前还有量的 IP。
2. AI 生成的分镜永远只算草稿，必须逐镜对标原视频，人工修正人物关系、镜头方向、情绪过渡和缺失镜头。
3. 先完成第一条成片，再围绕数据迭代；不要等到“全懂了”才发布，因为平台和模型都在快速变化。

### 最大一个坑：
- 坑点：看到一个热门形象或故事就直接开做，没有先验证当前流量，也没有先筛掉文化差异大或高风险的脚本，结果很容易零播放、白花时间和生成成本。
- 解决方案：把“IP 当月流量验证 + 脚本四条红线筛查”做成前置闸门，只有同时过关的 IP 和脚本才能进入分镜、生成和剪辑环节。

### 最值钱一句忠告：
“不要去猜什么火，让平台告诉你什么火。”先花 2 小时像普通用户一样刷 YouTube 无痕首页，把平台主动推给你的热门 IP 和爆款脚本截图存档，它们就是免费的选题池。

- 【推测】Seedance 的实际月度花费、单条视频平均成本、发布频率和起号节奏，原帖未披露。
- 【推测】第三层“深度围绕 IP 演绎”的具体 SOP 没有展开，当前案例更适合作为“前两层真人 AI 复刻”的操作样板。
