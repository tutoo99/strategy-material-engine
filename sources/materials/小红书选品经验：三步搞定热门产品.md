dark:prose-invert outline-none dark:!bg-[#20242A] box-border milkdown-theme-nord editor ProseMirror-focused"><p>大家好，我是银河。我在准备分享 小红书选品 内容的时候，在思考自己这几年的电商经验最重要的选品标准是什么。</p><p><br><br class="ProseMirror-trailingBreak"></p><p>因为我们实践的平台在变动，淘宝、抖音、视频号、小红书等等；平台的规则在变化，流量分配的调整、运营规则的提高等等；产品生命周期是会衰退的，热度的流失；竞争对手的增多等等。</p><p><br><br class="ProseMirror-trailingBreak"></p><p>平台的变动，规则的变化，产品的衰退，都会使我们不断地去选品，也会使选品的判断数据不断变动。记得前几年生财做过一次统计，说生财搜索次数最高的词是选品，可见大家对选品的需求很大。所以我在想，有没有一个标准，是不受这些变化而改变的。</p><p><br><br class="ProseMirror-trailingBreak"></p><p>我认真回想了我做淘宝、亚马逊的时候，是怎么考虑这个产品能不做的。<strong>第一个念头是 我能不能超过他做到第一名，第二个念头是 值不值得我去做。</strong></p><p><br><br class="ProseMirror-trailingBreak"></p><p>做淘宝的时候，用价格货源、付款人数、收货人数、晒图数量、问大家数量、运营手段、主图美观度等等来判断 我能不能超过他做到第一名。用访客、销量前五的收货梯队差、同链接付款人数、收货人数差值、竞品数量、大盘热度、竞品数量、大盘热度算出月利润、市场竞争、产品周期，这三个因素来判断 值不值得我去做。</p><p><br><br class="ProseMirror-trailingBreak"></p><p>那么换成小红书平台，用 竞品的粉丝量、笔记总数、链接销量、货源情况 、笔记素材的质量、每天发笔记的数量 来判断 我能不能超过他做到第一名。用 竞品24小时加购人数、销量记录情况、30天爆款笔记数量、头部竞品数量 来算出月利润、市场竞争、产品周期，再判断 值不值得我去做。</p><p><br class="ProseMirror-trailingBreak"></p><h3 id="">千言万语，最终汇集成一句话：<strong>只选能争第一的热度产品。</strong></h3><h3 id="">选品的数值标准不断更新换代，唯有能争第一的选品思维贯穿始终。</h3><p><br class="ProseMirror-trailingBreak"></p><p><br class="ProseMirror-trailingBreak"></p><p>接下来开始分享我的小红书选品经验。</p><p><br class="ProseMirror-trailingBreak"></p><p><br class="ProseMirror-trailingBreak"></p><p><strong>第一步：找到 能争第一的产品</strong></em></p><p><br><br class="ProseMirror-trailingBreak"></p><p>一个产品的市场，最赚钱肯定是头部的那几个，我们只要盯着市场第一名去竞争，最差也能做到第四第五名。这也就是大家常说盯着千万赚百万，盯着百万赚十万的道理。</p><p><br><br class="ProseMirror-trailingBreak"></p><p>首先我们要学会 找产品，简单来说就是 让你看到产品 的方法。这个环节都是简单实操的东西。</p><p><br><br class="ProseMirror-trailingBreak"></p><p>小红书站内找产品的方法：</p><ol><div><li class="flex flex-column gap-2 items-start"><span class="flex items-center"><span class="text-nord8">1.</span></span><div class="min-w-0"><div style="white-space;"><p>类目榜单（千帆后台-数据中心-市场-笔记排行-优秀账号）</p></div></div></li></div><div><li class="flex flex-column gap-2 items-start"><span class="flex items-center"><span class="text-nord8">2.</span></span><div class="min-w-0"><div style="white-space;"><p>搜索商品（小红书首页-右上角放大镜-输入关键词-商品-销量优先）</p></div></div></li></div><div><li class="flex flex-column gap-2 items-start"><span class="flex items-center"><span class="text-nord8">3.</span></span><div class="min-w-0"><div style="white-space;"><p>手刷笔记（小红书首页-下滑-查看带货的笔记）</p></div></div></li></div><div><li class="flex flex-column gap-2 items-start"><span class="flex items-center"><span class="text-nord8">4.</span></span><div class="min-w-0"><div style="white-space;"><p>友好集市（小红书首页-我-友好集市-下滑查看商品）</p></div></div></li></div></ol><p><br class="ProseMirror-trailingBreak"></p><p>其他平台找产品的方法：</p><ol><div><li class="flex flex-column gap-2 items-start"><span class="flex items-center"><span class="text-nord8">1.</span></span><div class="min-w-0"><div style="white-space;"><p>淘宝热搜榜单（淘宝首页-搜索框-榜单）</p></div></div></li></div><div><li class="flex flex-column gap-2 items-start"><span class="flex items-center"><span class="text-nord8">2.</span></span><div class="min-w-0"><div style="white-space;"><p>抖音商品榜单（抖音首页-我-创作者中心-全部-电商带货）</p></div></div></li></div><div><li class="flex flex-column gap-2 items-start"><span class="flex items-center"><span class="text-nord8">3.</span></span><div class="min-w-0"><div style="white-space;"><p>微博热搜榜单（微博首页-发现-更多热搜）</p></div></div></li></div><div><li class="flex flex-column gap-2 items-start"><span class="flex items-center"><span class="text-nord8">4.</span></span><div class="min-w-0"><div style="white-space;"><p>身边看到的所有产品</p><p><br class="ProseMirror-trailingBreak"></p></div></div></li></div></ol><p>接着我们要判断 我所看到的产品，我能不能争第一。先梳理出自己的电商运营能力，然后把竞品简单分下类，就可以判断出能否争第一了。</p><p><br class="ProseMirror-trailingBreak"></p><p>梳理自己的电商运营能力</p><ol><div><li class="flex flex-column gap-2 items-start"><span class="flex items-center"><span class="text-nord8">1.</span></span><div class="min-w-0"><div style="white-space;"><p>产品差异化：低价、产品升级（价格差异化需要供应链资源、产品升级需要设计开品的能力）</p></div></div></li></div><div><li class="flex flex-column gap-2 items-start"><span class="flex items-center"><span class="text-nord8">2.</span></span><div class="min-w-0"><div style="white-space;"><p>素材差异化：笔记素材的升级（针对竞品素材形式单一或不多的情况下，用更多的爆款形式去竞争）</p></div></div></li></div><div><li class="flex flex-column gap-2 items-start"><span class="flex items-center"><span class="text-nord8">3.</span></span><div class="min-w-0"><div style="white-space;"><p>场景搭建：能否找到竞品相似的场景</p></div></div></li></div><div><li class="flex flex-column gap-2 items-start"><span class="flex items-center"><span class="text-nord8">4.</span></span><div class="min-w-0"><div style="white-space;"><p>拍摄能力：能否拍出竞品同质量的图片</p></div></div></li></div><div><li class="flex flex-column gap-2 items-start"><span class="flex items-center"><span class="text-nord8">5.</span></span><div class="min-w-0"><div style="white-space;"><p>日更笔记数量：能否做到 日更竞品1.5-2倍的笔记数量</p></div></div></li></div></ol><p>根据竞品的账号数据，分成4个级别：</p><p><br class="ProseMirror-trailingBreak"></p><p>A类产品：24小时加购＞100 ； 账号粉丝＜200 ；发布笔记数量＜200 ； 日更笔记数量＜10 ；商品销量＞1000；</p><p>B类产品 ：24小时加购＞100 ； 账号粉丝＜1000 ；发布笔记数量＜500 ； 日更笔记数量＜10 ；商品销量＞1000</p><p>C类产品：24小时加购＞200 ； 账号粉丝＜2000 ；发布笔记数量＜1000； 日更笔记数量＜10 ；商品销量＞3000</p><p>D类产品：24小时加购＞500 ； 账号粉丝 不限 ；发布笔记数量 不限 ； 日更笔记数量＜20 ；商品销量＞10000</p><p><br><br class="ProseMirror-trailingBreak"></p><p>做A类产品，你必须满足③④</p><p>做B类产品，你必须满足③④⑤</p><p>做B类产品，你必须满足②③④⑤</p><p>做D类产品，你必须满足④⑤，且①或②需满足一个</p><p><br><br class="ProseMirror-trailingBreak"></p><p>24小时加购代表它是近期的热门产品；账号粉丝和发布笔记数量代表竞品的账号权重；日更笔记数量代表竞品的基础运营能力；商品销量代表它不是偶然性。</p><p><br><br class="ProseMirror-trailingBreak"></p><p>这种选品的初步筛选，你再看到竞争账号的第一眼就能判断出来自己能不能去争第一名了。</p><p><br class="ProseMirror-trailingBreak"></p><p><br class="ProseMirror-trailingBreak"></p><p><strong>第二步：选出 值得做的产品</strong></em></p><p><br class="ProseMirror-trailingBreak"></p><p>当我们找到 能争第一的产品后，我们还要看它值不值得我们去做。</p><p>什么是值得呢？我个人认为是：利润符合预期，竞争不激烈，入场时机好。</p><p><br><br class="ProseMirror-trailingBreak"></p><p>1、利润符合预期</p><p>小红书电商的笔记带货，60-80%的销量来自于爆款笔记，剩下的20-40%都来源于那些近期发布的个位数赞笔记。所以在我们还没有爆款笔记＜竞品的时候，我们只能做到他的20-40%的销售额。这也是为了尽可能保证，我们及时干不过第一名，那也能做到他30%的销售额。</p><p><br class="ProseMirror-trailingBreak"></p><p>如果你的目标是每天1000利润，那对标账号至少要每日利润＞3000</p><p>如果你的目标是每天3000利润，那对标账号至少要每天利润＞9000</p><p><br class="ProseMirror-trailingBreak"></p><p>销量的记录方法：手动记录，链接今天的销量-链接昨天的销量=一天的销量</p><p>利润的计算方法：销量*（售价-找到进货平台同款的大致成本）=利润</p><p><br><br class="ProseMirror-trailingBreak"></p><p>2、竞争不激烈</p><p>竞争激不激烈主要是看竞争对手的数量，我对竞争对手的定义就是 他能不能抢走我的单量。如果要转化成数值判断的话，就是每天发这个产品5篇以上笔记的账号，就是我的竞品。初步标准是：同款产品的有效竞品 ≤ 5家。</p><p><br><br class="ProseMirror-trailingBreak"></p><p>3、入场时机好</p><p>每个产品在一年中都有需求起伏的，一般分为季节性产品和非季节性产品。</p><p>季节性产品：一年中只在某个时间段有强需求，其他时间段需求低。（如：棉袄、驱蚊手环）</p><p>非季节性产品：全年都有需求（如：灯具 洗发水）<br><br class="ProseMirror-trailingBreak"></p><p>季节性产品入场时机最好在高峰期之前一个月，非季节性产品随时都可以。</p><p><br class="ProseMirror-trailingBreak"></p><p>经过生财小伙伴阿兜的提议，我把小红书电商选品的数值评判标准都梳理了出来，根据每个环节的重要程度进行比例划分，整理成了 小红书选品评分表。填入数据，就能得出这个产品的分数，满分10分，8分为及格。<br><br class="ProseMirror-trailingBreak"></p><p>简单来说就是，请了 表格化的我 来帮你判断这个产品能不能做，我不能保证8分以上产品的成功率，但8分以下一定是不值得做的。</p><p><br class="ProseMirror-trailingBreak"></p><p>当你选出能争第一的热度产品后，最后一步其实比较简单了。就是找到同货源或者同等质量的相似货源。</p><p>1、下单竞品链接</p><p>2、1688平台以图搜图，采购多家货源（SKU相同的情况下，尽量采购多个城市的货源）</p><p>3、货源和竞品对比（同货源是最好的，没有的话也要是外观相似且质量差不多的货源）</p><p><br><br class="ProseMirror-trailingBreak"></p><p>最后货源没有问题的情况下，选择 竞品最弱&amp;利润客观 的产品去实践。</p><p><br><br class="ProseMirror-trailingBreak"></p><p>以上是我小红书选品的所有经验分享，希望对大家有一丢丢丢的帮助。</p><p><br
