# 一、B站带货为什么能挣到钱？

**是平台释放的红利。**

哔哩哔哩开了这个带货的功能，视频下方就可以带链接，链接跳转京东淘宝用户，产生消费就会有佣金。

![](images/家蒙在生财分享哔哩哔哩-image-6.png)



我们看到的案例，有做的很差的视频，挂了链接也可以开单。（生财里也有这样的案例）

做的好的视频，解决了用户的选购消费困惑的问题，自然也有更好的开单效率。

其实不是我们有多厉害，也不是我们的视频做的有多漂亮，更加不是说，我们解决了用户多大的一个问题。

说的露骨一点，**就算没有我们，用户的问题，大概率自己也会解决掉。**

我们只是出现在这个位置上，他消费的时候经过了我们一手，这就产生了中间服务费。

能挣到这个钱的本质原因是，**现在是好时候， B站在释放红利。**



开了这个功能，能产生利润的机会，这些广告位，还没有被塞满。

**平台还很干净，没有被污染，用户还相信这一套。**

想起一句话，<u>强者从不抱怨环境，因为他们就是污染环境的那批人。</u>



再啰嗦一句，我们能挣钱是因为我们在正确的时候，出现在了这个位置上。

是，时机。

而不是我们有多厉害，有多强大。

如果过了两三年之后，B站的开单效率减变差，用户也觉得厌烦，不再相信我们推荐好物这一套。

那时候我们就可以说B站已经被污染了~~，而我们就是污染大环境的那帮人~~。



另外，都是带货，为什么我们做知乎带货和B站带货，却没有碰抖音快手小红书这些。

有一个很重要的区别，知乎和B站都是把自己的流量导出去，给天猫淘宝，京东。

而抖音、快手、小红书他们更封闭，更愿意把流量留在自己的平台内部，

所以他们更多是抖音小黄车、小红书店铺、抖店，这样的购物氛围是没有纯购物平台来的直接的。

后者更多是走，一件代发赚差价的逻辑。

前者是把流量导给成熟的购物平台，卖好货，卖品牌货，金额大，赚取微薄返佣的逻辑。

虽然大家都叫做带货，

但，**赚的是不同的钱。**

# 五、具体怎么上手？

好，看到这里，你说你看懂了，也大概知道有哪几类形式了。

那么接下来就是具体一点的，到底怎么上手。

视频的载体比起图文载体，展现形式是从2元变成4元，难度翻了4倍。所以很多圈友觉得难，不知道怎么下手，这是正常的。

也很高兴有这么一个门槛，他是拦路虎，也是护城河。

**进不去嘛，就是拦路虎；进去了，就是护城河。**



做视频这个事儿，拆解出来流程分为：

<span style="color: inherit; background-color: rgba(255,246,122,0.8)">选题——文字脚本——露脸拍摄——拍物品、找素材——整合剪辑，加声音特效——动画建模——发布运营</span>

<span style="color: inherit; background-color: rgba(255,246,122,0.8)">再复杂一点的，甚至涉及到策划、商务、样品采购等等。</span>



每一个环节都可以说十分重要，都可以占用掉大量的精力和时间。

吓不吓人？

一抓一大把，无从下手对不对。

但换个角度来想，其实这里面。**只要有几个关键的要素到位了，其他的都可以是0分。**

想想刚才的垃圾堆风格视频，人家没露脸也几乎没拍摄，更加谈不上剪辑，也没有声音特效动画建模。但就是奏效，就是有人买单。

**因为他确实解决了用户的问题。**

想要买一台笔记本电脑是一件。决策成本很高的事儿，不希望买错了来回退。

所以他看上去非常朴素，但是并没有影响他为用户**提供价值。**

我们自己做的会稍微认真一点，但是说起来，做了这么久，**也并非全掌握了当中的全部环节。**

所以一方面吓人是吓人，但考虑到不需要全都掌握，心理上的压力一下就少了很多。



具体一点来说，如果不想要认真准备，摆着个镜头，不论露脸不露脸，张嘴就来，现场直播，临场发挥就可以。

如果想要认真准备：

**第1步是脚本文案**，我们是干知乎出身的，文案这一块虽然说不是问题，但我们也有缺点，就是我们的废话太多了。不适合视频风格，反倒是在切换平台的过程中，要控制自己，不能说太多。

**第2步是拍摄**，这个是挺关键的一部分，涉及到人物画面，还有物体本身。

专业一点的术语叫做Aroll、Broll，人脸的部分叫做Aroll，物品的部分穿插的画面叫做Broll。

视频剪辑，讲究一个画面的切换，<u>央视的规则是三秒钟必须切换一个镜头</u>，所以他们有非常多的摄像机位置，也叫做机位。

我们就一个镜头，所以用不同的产品画面来遮盖人脸，在人物和产品之间来回跳跃。**这是一个让视频显得不那么枯燥的方法。**

但是我说的这些都是表面的技巧。不管你的画面有多么的有趣跳跃，用户看着一点也不疲劳。但你的内核你的文案，你想表达的东西是没有价值的，看这样的视频也是浪费时间。



下面这2个，都是专业人士眼中，表现力、灯光、画面都不太好的视频。

但他们在自己的领域里传达的信息和观点是非常有价值的，因此咱们这两个账号也做得非常好。

实质大于形式，就是这个意思：

# 六、设备操作简析

## 1、硬件

好理论知识讲完了，也许你也有那么一点信心，想要跃跃欲试，紧接着大家就会遇到设备的问题。

我是做播客的，很多人问我播客应该买什么设备，我就一直说，**新手做播客别买任何设备，你手上那一台最好的手机拿来用就足够了。**

现在做B站也是同样的道理，理论上一台手机就够用了，像我刚才上面两张图片，你觉得他们拍的画面很牛逼吗？

像素清晰吗？光线充足吗？背景干净吗？

不重要，对不对，不重要。

最多最多，弄一个<u>便宜的无线麦克风</u>，这个东西在淘宝两位数能买到，闲鱼买更便宜，

因为相机距离，嘴巴的距离稍微远，收音可能不太好。

**如果做一个项目忍不住想要花点钱，低于100块钱的无线麦克风，是我最推荐你购买的东西。**

再贵的，我就一律不建议购&#x4E70;**，至于设备升级，我们的原则是，<u>你升级的设备必须是你赚到的钱</u>，**

这样就不会吃灰，不会有浪费钱的内疚感。



三脚架、电灯泡这些就不说了，默认大家都有，至于怎么使用也没必要讲，自己喜欢，自己舒服就行。

千万不要觉得我在敷衍你，我们做过测试，

**认真做的，单反相机拍出来的视频，有时候带货效率比不上那些，画面抖动的、带有噪点的、手机拍摄的视频。**



## 2、软件

软件方面默认统一用剪映，虽然有很多剪辑软件，虽然剪映也要花钱。但这是市面上功能最齐全，并且最符合国人使用手感的一款软件。对于小白用户来说，上手难度相对较低。

这里也不想展开剪辑技巧，**剪辑原则就是一个有基础的形状和样子，哪怕有很多瑕疵也没有关系。**

B站是一个开放的平台，是一个神经病集中地，大家不必太正常。

**目的就是把一个视频做出来，忘记那些灯光不仅仅剩这些枷锁和束缚，先做东西再优化。**

**先做一个垃圾出来。**



## 3、规模

这里引申两个概念：

第1个叫**规模效应**；

第2个叫**E=核心算法 x 大量重复动作²**。

做第1个视频往往觉得很难，花了很长时间做出来一个垃圾，播放量也很差，是最打击人信心的。

但是做多了之后，会产生规模效应。

你会越来越熟练，**你会觉得做一个视频出来也不过如此。**

这个时候视频制作成本和心理门槛都会下降。

**想要做好这一件事情，一定要符合规模效应。**

浅尝辄止，玩了两下就跑了，是不可能有规模效应的。



# 七、好的视频，是什么感觉

刚才一直和大家灌输各种细节都不重要，事情没有想象的那么难。

但毕竟做还是想要做好，**我们不是无脑，也不是敷衍，最重要的是言之有物。**

一个好的视频，他大概是什么样的感觉呢？

他就像你邀请朋友来家里吃一顿饭。你为什么闲来无事想要请客，你想要准备什么样的菜品，让对方感到开心？

你是应该用精湛的技巧，追求满汉全席，还是追求有温度的家常小炒菜，哪怕盐放多了，水放多了，大家也能一起吃得很开心。

做一个视频和邀请朋友来家里吃饭的感觉，是一样的。

关于选品这里就不展开讲了，一方面讲多了我怕你抄我的，影响我搞钱。

另一方面，我请朋友吃饭，总是做辣椒炒肉，是因为我擅长，我喜欢，我的朋友也喜欢。

那你请朋友吃饭是不是也非得做辣椒炒肉呢？

也许你擅长牛排，那你就做牛排，**去寻找你擅长的有分享欲望的东西，比较重要。**



所以我们一直说对标，对标，你究竟是要对标它的<u>视频制作水准</u>，还是他的<u>精良画面</u>，还是要模仿他的<u>选品</u>，又或是是想要模仿他那样<u>真诚分享的感觉呢？</u>

**很难回答对不对，这件事情之所以有难度，就是因为正确答案太多了。**

要我说，B站能做出成绩来，你的发心，想要为别人解决问题的那个冲动，提供价值的这种出发点，是必不可少的。

毕竟我们不是说勉强地、忍耐地做一个视频，出几单，开心一下就完事了。

我们是要做规模的，我们要做很多视频，要做很长时间，**如果你自己都不喜欢，那真的是无从谈起。**

如果不是很懂，那还可以学，

如果说提不起兴趣，那就真的不建议入局，很痛苦。

**我们赚钱应该要快乐，不应该捏着鼻子。**



# 八、知乎当年经历了什么阶段

知乎的红利是从2019年开始爆发的，持续到了2025年结束。

最开始知乎也缺内容。只要写了，只要发了，就比较容易开单。

那时候大家极力追求的是专业感，知乎氛围下的装X感，非得是某个专家资深大佬才能说话，说话才有分量。

**所以一开始只要你装腔作势，就比较容易开单，这样的状态持续了一年。**

第2年，开启了知+广告系统，进入了自然流量和付费流量的混乱期。

第3年，自然流量几乎没有，付费流量成为主流，<u>这个时候新玩家再入局，就已经感受不到甜头了。</u>

生财圈友，应该在第1年就有接触，哪怕信息慢一点，到第2年，怎么着都能知道这个事。

对于圈外人来说，知道知乎能挣钱这个事，估计就是第3年的事，这时候就算知道了，也很难做到。

**信息的流动是有延时的。**

第4年。付费流量越来越贵，品牌也难受，创作者也难受。更多的人开始追求**广告商单**，而非带货。

第5年知乎平台每况愈下，平台的广告越来越多，真实的有用的内容越来越少，用户流失比较厉害，别说带货了，就连甲方商单也都越来越少。

据我的了解，目前还剩下一批头部玩家苟延残喘，鸡头凤尾嘛，人家还是有饭吃，甚至过得比较滋润，我也是在这一年败下阵来，没再坚持下去。

**所以现在B站还有很多新人入局，用不了几天就能开单，真的是非常幸福。**

**想当年我在知乎开第1单，用了一个半月的时间。**

至于说B站现在处于什么阶段，B站这个平台又能够持续几年，这个话题也不想讨论。

没必要，就像开头说的，**热闹的时候，大家会聊很多，表达自己的看法，事后回头看都没意义。**

跟着趋势走比较好。



# 九、B站和知乎有什么异同

写太长了。

我也直接给答案吧，视频载体，尤其是露脸的形式，个人感觉比图文，比那种装腔作势的堆砌，**信任感要来得更高。**

&#x20;B站的转化率相当不错的，和知乎鼎盛时期很接近，这和露脸有很大关系。

再就是， B站整个平台氛围更加宽松，在这里可以骂人，可以Battle，不会动不动封你的号。

我知乎一个半月才开第1单，就是因为骂人，辛辛苦苦准备的第1个号被封了。

你猜我骂了啥——对方骂我没下限，我骂他阴阳怪气。**就这4个字，浪费了1个月时间。**

再就是，B站还挺开放的，每个月会公布榜单，这个是知乎从来没有过的。**可见阿B也希望大家互相学习借鉴卷起来，为他这个平台创造更大的价值。**

阿B很追求进步的，他想要规模，想要声势，想要更多人参与。

你知道吗，在B站，带货的佣金比例，创作者可以拿到100%。

而知乎只能给到大家70%，他自己还想要30%的参与费呢。

&#x20;B站在这一点上显得特别不同，**他一分钱都不要，给大家提供平台，给大家提供流量，大家只要做好内容就行了。阿B只要大家来玩\~**

但从缺点上来说，知乎是一个有长尾流量的平台。

**说句不好意思的，我现在赚的一些钱，还是5年前写的文章给我带来的，知乎长尾的感觉真的很美妙。**

在B站上视频平台，确实没有这么长的长尾效应。
